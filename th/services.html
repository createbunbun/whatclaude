💌 คล็อดแชท

นี่คือการใช้งานขั้นพื้นฐานที่สุด เพียงพูดคุยกับ Claude จากเบราว์เซอร์หรือแอปสมาร์ทโฟนของคุณ คุณก็สามารถสนทนากับ AI ได้ทันที คุณสามารถเริ่มต้นได้ฟรีโดยการสร้างบัญชี

แพลตฟอร์มที่รองรับ 🌐 เบราว์เซอร์ (พีซี/สมาร์ทโฟน)
📱 แอป iOS/Android
🖥️ แอปเดสก์ท็อป Mac
🪟 แอปเดสก์ท็อป Windows
แผนที่คุณต้องการ มีให้จากแผนฟรี

วิธีการเริ่มต้น:claude.ai เพียงเยี่ยมชมและลงทะเบียนด้วยที่อยู่อีเมลหรือบัญชี Google ของคุณ สิ่งที่คุณต้องทำคือพิมพ์คำถามของคุณลงในช่องป้อนข้อมูลแล้ว Claude จะตอบคำถามให้คุณ

คุณสามารถทำได้:

  • การสร้างและแก้ไขข้อความ (อีเมล รายงาน บทความในบล็อก ฯลฯ)
  • คำตอบสำหรับคำถาม (ช่วยในการวิจัย)
  • การอภิปราย/ให้คำปรึกษาเกี่ยวกับไอเดียวอลล์
  • การแปล/บทสรุป
  • การคำนวณและการจัดระเบียบข้อมูล

📁 โครงการ

คุณสมบัตินี้ช่วยให้คุณจัดระเบียบการสนทนาตามธีมได้ คุณสามารถค้นหาการสื่อสารที่ผ่านมาได้อย่างง่ายดายโดยแบ่งออกเป็นหมวดหมู่เช่น "งาน" "การเรียนภาษาอังกฤษ" และ "การวางแผนการเดินทาง" คุณยังสามารถป้อนคำแนะนำแบบกำหนดเอง (คำแนะนำที่ตั้งไว้ล่วงหน้า) สำหรับแต่ละโปรเจ็กต์ได้

แพลตฟอร์มที่รองรับ 🌐 เบราว์เซอร์
📱 แอพสมาร์ทโฟน
แผนที่คุณต้องการ มีให้จากแผนฟรี

มันเหมือนกับโฟลเดอร์การสนทนา ตัวอย่างเช่น หากคุณตั้งค่าคำสั่งสำหรับโครงการ ``งาน'' เป็น ``โปรดใช้ภาษาสุภาพในอีเมลธุรกิจ'' Claude จะตอบกลับด้วยน้ำเสียงนั้นโดยไม่ต้องระบุทุกครั้ง คุณยังสามารถอัปโหลดไฟล์และถามคำถามตามเนื้อหาได้

🖥️ โคเวิร์ค

คุณสมบัตินี้ช่วยให้ Claude ทำงานเบื้องหลังบนคอมพิวเตอร์ของคุณโดยอัตโนมัติ ตัวอย่างเช่น คุณสามารถพูดว่า "เก็บเอกสารนี้ไว้ด้วยกัน" และ Claude จะทำงานเบื้องหลังในขณะที่คุณทำงานอย่างอื่น

แพลตฟอร์มที่รองรับ 🖥️ Mac / 🪟 แอป Windows Desktop
แผนที่คุณต้องการ โปรขึ้นไป
⚠️ บันทึก:คุณจะต้องติดตั้งแอปเดสก์ท็อป (เวอร์ชัน Mac หรือ Windows) ไม่สามารถใช้ในเวอร์ชันเบราว์เซอร์ (claude.ai) หรือแอปสมาร์ทโฟน

“ตัวแทน” คืออะไร? "ตัวแทน" ในที่นี้หมายถึง AI ที่คิดด้วยตัวเองและดำเนินงานในนามของคุณ การแชทปกติเกี่ยวข้องกับการไปกลับของ "คำถาม → คำตอบ" แต่เจ้าหน้าที่จะดำเนินการหลายขั้นตอนในขณะที่ทำการตัดสินใจด้วยตนเอง ภาพที่เห็นคือ "ผู้ช่วย AI" กำลังทำงานอยู่เบื้องหลัง

💻 คล็อดโค้ด

เป็นเครื่องมือ AI ที่เชี่ยวชาญด้านการเขียนโปรแกรม (การพัฒนาซอฟต์แวร์) คุณสามารถปล่อยให้การสร้าง การแก้ไข และการดีบักโค้ดของคุณ (การตรวจสอบและแก้ไขสาเหตุของข้อผิดพลาด) เป็นหน้าที่ของ Claude

แพลตฟอร์มที่รองรับ 🖥️ Mac / 🪟 Windows / 🐧 Linux
(ต้องติดตั้งบนเดสก์ท็อป)
แผนที่คุณต้องการ แผนสูงสุดหรือ API จ่ายตามการใช้งาน
⚠️ บันทึก:จำเป็นต้องติดตั้ง ทำงานบนหน้าจอที่เรียกว่า "Terminal" เครื่องมือสำหรับโปรแกรมเมอร์และวิศวกร

“เทอร์มินัล” คืออะไร? หน้าจอสำหรับออกคำสั่งข้อความไปยังคอมพิวเตอร์ ไม่มีปุ่มหรือเมนูเหมือนในแอปที่คุณมักจะใช้ และคุณควบคุมได้โดยการป้อนคำสั่ง (คำแนะนำ) บนแป้นพิมพ์ นี่คือ ``Terminal'' บน Mac และ ``Command Prompt'' หรือ ``PowerShell'' บน Windows

Claude Code ทำงานบนเทอร์มินัลนี้และอ่านไฟล์ของโปรเจ็กต์ของคุณในขณะที่คุณเขียนและแก้ไขโค้ดของคุณ

มันเหมือนกับ "สูตร" ในการทำอาหาร ตัวอย่างเช่น หากคุณบันทึกชุดของขั้นตอน เช่น ``เรียกใช้การทดสอบและแก้ไขหากล้มเหลว'' ไว้เป็นทักษะ คุณสามารถดำเนินการได้โดยอัตโนมัติในครั้งถัดไปด้วยคำสั่งสั้นๆ เพียงคำสั่งเดียว นี่เป็นคุณสมบัติที่มีประโยชน์สำหรับผู้ที่ใช้ Claude Code อยู่แล้ว

🔌 บูรณาการ

คุณลักษณะที่ช่วยให้ Claude สามารถทำงานร่วมกับบริการและเครื่องมืออื่นๆ ได้ โหลดไฟล์จาก Google Drive หรือใช้ผลการค้นหาเว็บเพื่อตอบคำถามของคุณ

แพลตฟอร์มที่เข้ากันได้ (แตกต่างกันไปขึ้นอยู่กับพันธมิตร)
  • บูรณาการ Google ไดรฟ์ 🌐 เบราว์เซอร์
  • ค้นหาเว็บ 🌐 เบราว์เซอร์ / 📱 แอป
  • ส่วนขยายของ Chrome 🌐 เบราว์เซอร์ Chrome เท่านั้น
  • การเชื่อมโยง Excel/PowerPoint 🪟 วินโดว์ / 🖥️ แมค
  • บูรณาการหย่อน บนหย่อน
  • MCP (โปรโตคอลการเชื่อมต่อเครื่องมือภายนอก) ในโค้ดของคลอดด์ (🖥️ Mac / 🪟 Windows / 🐧 Linux)
แผนที่คุณต้องการ แตกต่างกันไปขึ้นอยู่กับฟังก์ชั่น (หลายรุ่นเป็น Pro หรือสูงกว่า)

สิ่งที่คุณไม่สามารถทำกับ Claude เพียงอย่างเดียวก็สามารถทำได้ด้วยส่วนขยาย ตัวอย่างเช่น หากคุณเปิดการผสานรวม Google ไดรฟ์ คุณจะสามารถถามคำถามกับ Claude เกี่ยวกับเนื้อหาในเอกสารและสเปรดชีตในไดรฟ์ได้

🔍API

นี่เป็นกลไกสำหรับผู้ที่สร้างแอปและบริการของตนเองเพื่อเรียกใช้ฟังก์ชันของ Claude จากภายในโปรแกรมของตน

แพลตฟอร์มที่รองรับ โทรจากโปรแกรม (ไม่คำนึงถึง OS)
แผนที่คุณต้องการ Pay-as-you-go (จ่ายเฉพาะสิ่งที่คุณใช้)

"API" เป็นเหมือนหน้าต่างสำหรับแลกเปลี่ยนข้อมูลระหว่างแอปพลิเคชัน ตัวอย่างเช่น หากคุณต้องการฝัง "ฟีเจอร์แชท AI" บนเว็บไซต์ของคุณ คุณสามารถใช้ API เพื่อส่งคำถามถึง Claude และรับคำตอบได้ ต้องมีความรู้ด้านการเขียนโปรแกรม และค่าธรรมเนียมจะขึ้นอยู่กับปริมาณที่ใช้ ไม่ใช่สำหรับผู้ใช้ทั่วไป แต่สำหรับนักพัฒนาและวิศวกร

📊 ตารางเปรียบเทียบการบริการ

เปรียบเทียบคุณลักษณะของแต่ละบริการได้อย่างรวดเร็ว

ชื่อบริการ ในหนึ่งคำ สภาพแวดล้อมที่เข้ากันได้ แผนที่จำเป็น สำหรับคนแบบนี้
💌 แชท พูดคุยกับเอไอ เบราว์เซอร์ / แอพสมาร์ทโฟน / Mac / Windows ฟรี~ ผู้ใช้ Claude ครั้งแรก
📁 โครงการ จัดระเบียบการสนทนาตามธีม แอพเบราว์เซอร์/สมาร์ทโฟน ฟรี~ ผู้ที่ต้องการใช้เพื่อวัตถุประสงค์หลายประการ
🖥️ โคเวิร์ค มันทำงานอยู่เบื้องหลังโดยอัตโนมัติ แอปเดสก์ท็อป Mac/Windows โปรขึ้นไป ผู้ที่ต้องการทำให้พีซีทำงานอัตโนมัติ
💻 คล็อดโค้ด การเขียนโปรแกรม AI Mac/Windows/Linux (เทอร์มินัล) การเรียกเก็บเงินสูงสุด / API วิศวกรโปรแกรมเมอร์
⚡ทักษะ ทางลัดขั้นตอนการทำงาน ในรหัสคลอดด์ การเรียกเก็บเงินสูงสุด / API ผู้ที่ต้องการปรับปรุง Claude Code
🔌 ส่วนขยาย ความร่วมมือกับบริการอื่น ๆ เบราว์เซอร์/บริการ ขึ้นอยู่กับฟังก์ชั่น ผู้ที่ต้องการรวมเข้ากับเครื่องมือภายนอก
🔍API สร้าง AI ลงในแอปของคุณ โทรจากโปรแกรม จ่ายตามที่คุณไป ผู้พัฒนา/บริษัท
