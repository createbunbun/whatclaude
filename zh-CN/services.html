💬 克劳德聊天

这是最基本的用法。 只需通过浏览器或智能手机应用程序与克劳德交谈，您就可以立即与人工智能进行对话。 您可以通过创建帐户免费开始。

📱 iOS/Android 应用程序
🖥️ Mac 桌面应用程序
🪟 Windows 桌面应用程序
您需要的计划 可从免费计划获得

如何开始：claude.ai 只需访问并使用您的电子邮件地址或 Google 帐户注册即可。 您所要做的就是在输入字段中输入您的问题，克劳德将为您解答。

你可以这样做：

  • 创建和更正文本（电子邮件、报告、博客文章等）

  • 创意墙讨论/咨询
  • 翻译/摘要
  • 计算和数据组织

🖥️ 协同办公

此功能允许克劳德自动在您的计算机上进行幕后工作。 例如，您可以说“将此文档放在一起”，克劳德将在您处理其他事情时在后台处理它。

支持的平台 🖥️ Mac / 🪟 Windows 桌面应用程序
您需要的计划 专业版及以上
⚠️ 笔记：您需要安装桌面应用程序（Mac 或 Windows 版本）。 它不能在浏览器版本（claude.ai）或智能手机应用程序中使用。

什么是“代理人”？ 这里的“代理”是指能够独立思考并代表你执行任务的人工智能。 正常的聊天涉及“问题→答案”的往返，但代理在做出自己的决定时会经历多个步骤。 形象地说，有一个“AI助手”正在幕后工作。

💻 克劳德代码

它是专门用于编程（软件开发）的AI工具。 您可以将代码的创建、修改和调试（调查并纠正错误原因）留给 Claude。

支持的平台 🖥️ Mac / 🪟 Windows / 🐧 Linux
（需要安装在桌面上）
您需要的计划 最大套餐或 API 即用即付
⚠️ 笔记：需要安装。 在称为“终端”的屏幕上进行操作。 程序员和工程师的工具。

什么是“终端”？ 用于向计算机发出文本指令的屏幕。 没有像您通常使用的应用程序那样的按钮或菜单，您通过在键盘上输入命令（指令）来操作它。 在 Mac 上这是“终端”，在 Windows 上是“命令提示符”或“PowerShell”。

Claude Code 在此终端上运行，并在您编写和修改代码时读取项目文件。

⚡技能

快捷功能可让您保存常用的工作程序并通过单个命令调用它们。 这省去了您一遍又一遍输入相同指令的麻烦。

支持的平台 在克劳德代码中使用
（🖥️ Mac / 🪟 Windows / 🐧 Linux）
您需要的计划 与 Claude Code 相同（Max 套餐或 API 即用即付）

这就像烹饪中的“食谱”。 例如，如果您将“运行测试并在失败时修复它”等一系列步骤保存为技能，则下次可以使用单个简短命令自动执行它。 对于那些已经在使用 Claude Code 的人来说，这是一个有用的功能。
