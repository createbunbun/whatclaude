Quickstart

APIの始め方

1

コンソールアカウントを作成

console.anthropic.comでAnthropicコンソールアカウントを作成。claude.aiアカウントとは別。

2

APIキーを発行

コンソール → API Keys → 「Create Key」で発行。sk-ant-で始まるキーを安全に保管。

3

SDKをインストール

Node.js: npm install @anthropic-ai/sdk　Python: pip install anthropic

4

最初のリクエストを送る

APIキーを環境変数 ANTHROPIC_API_KEY に設定し、下記のコードを実行。

5

本番環境へ

レート制限・エラーハンドリング・プロンプトキャッシュを設定してデプロイ。

Examples

コードサンプル

JavaScript / Node.js — 基本的なメッセージ
import Anthropic from '@anthropic-ai/sdk'; const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY }); const message = await client.messages.create({ model: "claude-sonnet-4-6", max_tokens: 1024, messages: [{ role: "user", content: "Pythonでフィボナッチ数列を生成して" }] }); console.log(message.content[0].text);
Python — ストリーミングレスポンス
import anthropic client = anthropic.Anthropic() with client.messages.stream( model="claude-sonnet-4-6", max_tokens=1024, messages=[{"role":"user","content":"長い物語を書いて"}] ) as stream: for text in stream.text_stream(): print(text, end="", flush=True)
