💻 Claude Code

Claude Code

コーディング専用の AI エージェント。ターミナル（コマンドライン）から動作します。

🤖
自律的なコーディング

ファイルの作成・編集・削除、コマンド実行、テストなどを Claude が自動で行います。「このアプリを作って」と指示するだけでコードを自律的に書き進めます。

🔍
コードベース理解

既存のプロジェクトフォルダを読み込んで、コード全体を理解した上でバグ修正・リファクタリング・機能追加ができます。

⚡
コマンド実行

npm install・git・テストスクリプトなどのコマンドを自動実行。開発フローをそのまま自動化できます（要確認）。

🌐
Web 検索・ドキュメント参照

最新のライブラリ情報やドキュメントを検索しながらコードを生成。最新情報をもとにした実装が可能です。

🚀 インストール・セットアップ

⚠️
Claude Code の利用には Anthropic API キー（有料）が必要です。API ガイド でキーの取得方法をご確認ください。

必要な環境

Node.js 18 以降 npm または npx macOS / Linux / Windows (WSL推奨) Anthropic API キー

セットアップ手順

1

Node.js をインストール

nodejs.org から LTS 版をダウンロードしてインストールします。

2

Claude Code をインストール

$ npm install -g @anthropic-ai/claude-code added 248 packages in 12s
3

API キーを環境変数に設定

# macOS / Linux $ export ANTHROPIC_API_KEY='sk-ant-...' # Windows PowerShell > $env:ANTHROPIC_API_KEY = 'sk-ant-...'
4

プロジェクトフォルダで起動

$ cd ~/projects/my-app $ claude ✓ Claude Code v1.x.x 作業ディレクトリ: /Users/you/projects/my-app 何をしましょうか？

🔌 IDE・エディタ連携

📝
VS Code

VS Code の統合ターミナルから Claude Code を起動すると、現在のファイルのコンテキストを自動で読み込みます。

💎
JetBrains IDEs

IntelliJ IDEA・PyCharm・WebStorm などの JetBrains 製品でも統合ターミナルからクロード Codeを利用できます。

🐙
GitHub との連携

GitHub Actions や GitHub CLI と組み合わせて、プルリクエストのレビューや自動コード修正も可能です。
