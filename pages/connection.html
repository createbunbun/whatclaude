🔌 MCP（Model Context Protocol）

Claude が外部データ・ツールにアクセスするための標準規格

❓
MCP とは？

Anthropic が策定したオープン規格。Claude（AIクライアント）が外部のデータソースやツールにセキュアにアクセスするためのプロトコルです。

🌟
何ができる？

Google Drive のファイルを読み込む・Slack にメッセージを投稿する・データベースを照会する、などを Claude が自律的に実行できます。

🔧
セットアップ方法

Claude Code または Claude デスクトップアプリの設定画面から MCP サーバーを追加・設定します。JSON 形式の設定ファイルで管理します。

🛠️ 主な連携サービス・ツール

📁
Google Drive / Docs

Google ドライブのファイル検索・読み込み・Docs の作成・編集。資料をアップロードせずに Claude が直接参照できます。

MCP 対応

💬
Slack

Slack のメッセージ検索・投稿・チャンネル管理を Claude が代行。業務自動化・議事録作成などに活用できます。

MCP 対応

🐙
GitHub

リポジトリの参照・プルリクエストのレビュー・Issue 管理を Claude が支援。Claude Code との組み合わせが効果的です。

MCP 対応

📊
Notion / Confluence

ドキュメント管理ツールとの連携。ページ作成・内容検索・更新を Claude が自動化します。

MCP 対応

🗃️
データベース

SQLite・PostgreSQL・MySQL などに MCP 経由で接続。自然言語でデータ照会・更新が可能になります。

MCP 対応

🌐
Web ブラウジング

Web 検索・ページ取得を Claude が自律的に実行。最新情報の収集・調査・レポート作成に活用できます。

Pro 以上

🔐 接続セキュリティ

通信のセキュリティ

  • 🔒
    すべての通信は HTTPS / TLS 1.3 で暗号化
  • 🔑
    API キーはリクエストヘッダーで送信（URL に含めない）
  • 🛡️
    Anthropic のインフラは SOC 2 Type II 認定取得

データプライバシー

  • 🚫
    デフォルトでは会話データはモデル訓練に使用されない（設定で変更可）
  • 📋
    Enterprise プランではカスタムデータ保持ポリシーを設定可能
  • 🌍
    GDPR 対応。EU ユーザーは EU 内でデータ処理可能
ℹ️
最新のプライバシーポリシーと利用規約は anthropic.com/privacy でご確認ください。

📡 ネットワーク要件

必要なインターネット接続

Claude はクラウドサービスのため、安定したインターネット接続が必要です。

  • ✓
    最小推奨帯域: 1 Mbps 以上
  • ✓
    接続先: api.anthropic.com / claude.ai
  • ✓
    プロキシ環境でも利用可能（HTTPS プロキシ対応）

オフライン利用について

現在、Claude はオフラインでの AI 推論には対応していません。ただし以下は可能です：

  • 📋
    過去の会話履歴の閲覧（アプリ版）
  • 📝
    保存済み Projects の内容確認
