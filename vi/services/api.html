Ví dụ cơ bản sử dụng SDK Python

1

Cài đặt SDK

cài đặt pip nhân loại
2

Đặt khóa API

xuất khẩu ANTHOPIC_API_KEY='your-api-key-đây'
3

gửi tin nhắn

nhập khẩu nhân loại client = anthropic.Anthropic() tin nhắn = client.messages.create( người mẫu="claude-sonnet-4-6", max_tokens=1024, tin nhắn=[ {"vai trò": "người dùng", "nội dung": "Xin chào!"} ] ) in(tin nhắn.content)

Ví dụ cơ bản sử dụng Node.js/TypeScript SDK

1

Cài đặt SDK

cài đặt npm @anthropic-ai/sdk
2

viết mã

nhập khẩu nhân loại từ '@anthropic-ai/sdk'; hằng số khách hàng = mới Nhân loại({ apiKey: tiến trình.env.ANTHROPIC_API_KEY, }); hằng số tin nhắn = chờ đợi client.messages.create({ mô hình: 'claude-sonnet-4-6', max_tokens: 1024, tin nhắn: [ { vai trò: 'người dùng', nội dung: 'Xin chào bằng tiếng Nhật! ' } ], }); console.log(message.content);

cuộn tròn https://api.anthropic.com/v1/messages \ --tiêu đề "x-api-key: $ANTHOPIC_API_KEY" \ --tiêu đề "phiên bản nhân loại: 2023-06-01" \ --tiêu đề "loại nội dung: ứng dụng/json" \ --dữ liệu '{ "người mẫu": "claude-sonnet-4-6", "max_token": 1024, "tin nhắn": [ { "vai trò": "người dùng", "nội dung": "Xin chào!" } ] }'

Các chức năng API chính

💬
API tin nhắn

API để gửi và nhận tin nhắn kiểu trò chuyện cơ bản. Hỗ trợ lời nhắc hệ thống và hội thoại nhiều lượt.

⚡
phát trực tuyến

Nhận một luồng phản hồi trong thời gian thực. Đây là phương pháp triển khai được đề xuất để cải thiện trải nghiệm người dùng.

🔧
Sử dụng công cụ

Một chức năng cho phép Claude gọi các công cụ và chức năng bên ngoài (Function Calling). Được sử dụng để tìm kiếm cơ sở dữ liệu, cộng tác API, v.v.

👁️
Tầm nhìn (đầu vào hình ảnh)

Một chức năng chuyển hình ảnh làm đầu vào và cho phép hiểu và phân tích nội dung. Tương thích với PNG, JPEG, GIF và WebP.

📦
Xử lý hàng loạt

API hàng loạt xử lý số lượng lớn yêu cầu không đồng bộ. Đạt được mức giảm chi phí và thông lượng cao.

💾
Bộ nhớ đệm

Khả năng lưu trữ các mã thông báo có cùng tiền tố để giảm chi phí và độ trễ. Hiệu quả đối với các lời nhắc hệ thống dài.
