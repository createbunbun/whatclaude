Что такое МКП?

Открытый стандарт, разработанный Anthropic. Протокол для Клода (ИИ-клиента) для безопасного доступа к внешним источникам данных и инструментам.

🌟
Что ты можешь сделать?

Клод может автономно читать файлы с Google Диска, публиковать сообщения в Slack, запрашивать базы данных и т. д.

🔧
Как настроить

Добавьте и настройте серверы MCP на экране настроек Claude Code или настольного приложения Claude. Управление с помощью файлов конфигурации формата JSON.

🛠️ Основные связанные сервисы и инструменты

📁
Google Диск/Документы

Поиск/чтение файлов на Google Диске, создание/редактирование документов. Клод может напрямую ссылаться на материалы, не загружая их.

Совместимость с MCP

💬
Слабый

Клод занимается поиском сообщений Slack, их публикацией и управлением каналами. Его можно использовать для автоматизации бизнеса, создания протоколов встреч и т. д.

Совместимость с MCP

🐙
GitHub

Клод помогает с просмотром репозитория, проверками запросов на включение и управлением проблемами. Эффективен в сочетании с Клод Код.

Совместимость с MCP

📊
Идея / Слияние

Интеграция со средствами документооборота. Claude автоматизирует создание страниц, поиск и обновление контента.

Подключитесь к SQLite, PostgreSQL, MySQL и т. д. через MCP. Данные можно проверять и обновлять, используя естественный язык.

    Все коммуникации есть HTTPS/TLS 1.3 зашифровано с помощью

📡 Требования к сети

Требуется подключение к Интернету

Claude — это облачный сервис, для которого требуется стабильное подключение к Интернету.

  • ✓
    Минимальная рекомендуемая пропускная способность: 1 Мбит/с или выше.
  • ✓
    Подключитесь к: api.anthropic.com/claude.ai
  • ✓
    Также может использоваться в среде прокси (совместим с HTTPS-прокси)
