Skills

Skills

A function that allows you to save frequently used tasks as "recipes" and recall them with one click.

What are skills?

Easy to understand even for first-time users

Something like a cooking “recipe”

Instead of having to explain the same steps every time, just select a saved skill and Claude will follow the same steps. For example, it is ideal for repetitive tasks such as "creating weekly reports" and "meeting minutes in a fixed format."

Recommended for these people

This feature is perfect for those who find it tedious to enter the same instructions every time, those who want to standardize work procedures within a team, and those who want to automate routine tasks and save time.

Supported platforms

Use within Claude Code

Skills are available in Claude Code's terminal environment. Compatible with Mac / Windows / Linux.

ℹ️
Requires installation of Claude Code. This function is for people who have programming knowledge.

Specific examples of skills

You can use it like this
📊
routine work

"Weekly report" skill

Compile weekly data in a fixed format. If you set the data aggregation method and output format in advance, you can complete the report with one click.

🔍
development

"Code review" skills

Check your code according to your team's coding rules. Check points such as naming conventions, error handling, performance, etc. can be defined in advance.

🌐
translation

"Translation check" skill

Compare the original text and the translated text and suggest improvements. It automatically checks for uniformity of terminology, accuracy of nuances, readability, etc.

📝
Git

"PR creation" skill

Automatically generates a pull request description from the commit content. Quickly create PRs with change summaries, test steps, review points, and more.

How to create skills

Easy to create in 3 steps
  1. 1

    Run the /skill command

    In Claude Code's terminal /skill Enter the command. The skill creation wizard will start.

  2. 2

    Describe the skill name and steps

    Name your skill and describe the steps you want Claude to perform. You can write detailed instructions in Markdown format.

  3. 3

    Just call me next time

    The skill you created /skillname You can call it by name like . No more explaining the same steps every time.
