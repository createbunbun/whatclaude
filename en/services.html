Claude's service list

An at-a-glance guide to see which service is right for you.
Organized by plan and supported environment.

💬 Claude Chat

This is the most basic usage. Just talk to Claude from your browser or smartphone app and you'll be able to have a conversation with the AI ​​right away. You can get started for free by creating an account.

Supported platforms 🌐 Browser (PC/Smartphone)
📱 iOS/Android app
🖥️ Mac Desktop App
🪟 Windows desktop app
Plan you need Available from free plan

How to get started:claude.ai Simply visit and register with your email address or Google account. All you have to do is type your question into the input field and Claude will answer it for you.

You can do this:

  • Creating and correcting texts (emails, reports, blog articles, etc.)
  • Answers to questions (help with research)
  • Idea wall discussion/consultation
  • Translation/Summary
  • Calculation and data organization

View chat details →

📁 Projects

This feature allows you to organize conversations by theme. You can easily find past communications by dividing them into categories such as "work," "English learning," and "travel planning." You can also enter custom instructions (preset instructions) for each project.

Supported platforms 🌐 Browser
📱 Smartphone app
Plan you need Available from free plan

It's like a folder of conversations. For example, if you set an instruction for your ``work'' project to ``Please use polite language in business emails,'' Claude will reply in that tone without having to specify it every time. You can also upload files and ask questions based on their content.

View details of Chat/Projects →

🖥️ Cowork

This feature allows Claude to automatically work behind the scenes on your computer. For example, you can say, "Keep this document together," and Claude will work on it in the background while you work on something else.

Supported platforms 🖥️ Mac / 🪟 Windows Desktop App
Plan you need Pro and above
⚠️ Note:You will need to install the desktop app (Mac or Windows version). It cannot be used in the browser version (claude.ai) or smartphone app.

What is an “agent”? The "agent" here refers to an AI that thinks on its own and carries out tasks on your behalf. Normal chat involves a round trip of "question → answer", but agents proceed through multiple steps while making their own decisions. The image is that an "AI assistant" is working behind the scenes.

See details about Cowork →

💻 Claude Code

It is an AI tool specialized in programming (software development). You can leave the creation, modification, and debugging of your code (investigating and correcting the cause of errors) to Claude.

Supported platforms 🖥️ Mac / 🪟 Windows / 🐧 Linux
(Requires installation on desktop)
Plan you need Max plan or API pay-as-you-go
⚠️ Note:Installation required. Operate on a screen called "Terminal". A tool for programmers and engineers.

What is a “terminal”? A screen for issuing text instructions to a computer. There are no buttons or menus like in the apps you usually use, and you operate it by entering commands (instructions) on the keyboard. This is ``Terminal'' on Mac, and ``Command Prompt'' or ``PowerShell'' on Windows.

Claude Code runs on this terminal and reads your project's files as you write and modify your code.

Learn more about Claude Code →

⚡Skills

A shortcut function that allows you to save frequently used work procedures and call them up with a single command. This saves you the trouble of entering the same instructions over and over again.

Supported platforms Use within Claude Code
(🖥️ Mac / 🪟 Windows / 🐧 Linux)
Plan you need Same as Claude Code (Max plan or API pay-as-you-go)

It's like a "recipe" in cooking. For example, if you save a series of steps such as ``Run a test and fix it if it fails'' as a skill, you can automatically execute it next time with a single short command. This is a useful feature for those who are already using Claude Code.

View details of Skills →

🔌 Integrations

A feature that allows Claude to work with other services and tools. Load files from Google Drive or use web search results to answer your questions.

Compatible platforms (varies depending on partner)
  • Google Drive integration 🌐 Browser
  • Web search 🌐 Browser / 📱 App
  • Chrome extension 🌐 Chrome browser only
  • Excel/PowerPoint linkage 🪟 Windows / 🖥️ Mac
  • Slack integration On Slack
  • MCP (External Tool Connection Protocol) In Claude Code (🖥️ Mac / 🪟 Windows / 🐧 Linux)
Plan you need Varies depending on function (many are Pro or higher)

Things you can't do with Claude alone can be done with extensions. For example, if you turn on the Google Drive integration, you'll be able to ask Claude questions about the contents of your documents and spreadsheets in Drive.

View extension details →

🔧API

This is a mechanism for people who are creating their own apps and services to call Claude's functions from within their programs.

Supported platforms Call from program (regardless of OS)
Plan you need Pay-as-you-go (pay only for what you use)

"API" is something like a window for exchanging data between applications. For example, if you want to embed an "AI chat feature" on your website, you can use the API to send questions to Claude and receive answers. Programming knowledge is required, and fees are based on the amount used. It's not for general users, but for developers and engineers.
