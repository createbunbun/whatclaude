Claude チャット

ブラウザやスマホから、誰でも無料で始められる Claude の基本機能です

Claude チャットとは

💬
Claude チャットは、claude.ai またはスマートフォンアプリから使える AI チャットサービスです。テキストボックスに質問を入力して送信するだけで、Claude が回答してくれます。アカウント登録は無料で、すぐに始められます。

対応プラットフォーム

プラットフォーム 対応環境
🌐 ブラウザ（claude.ai） Windows / Mac / Linux 全対応
📱 スマホアプリ iOS / Android
🖥️ デスクトップアプリ Mac / Windows

始め方（3ステップ）

1

アクセス・ダウンロード

claude.ai にアクセス、またはアプリをダウンロード

2

無料登録

メールアドレスまたは Google アカウントで無料登録

3

チャット開始

メッセージを入力して送信するだけ

できること

❓

質問・調べもの

「光合成の仕組みを中学生にわかるように教えて」のように、知りたいことを自然な言葉で質問できます。

✍️

文章作成

「取引先へのお詫びメールを書いて」など、ビジネス文書からカジュアルな文章まで作成をサポートします。

🌐

翻訳・校正

「この英文を自然な日本語に翻訳して」のように、多言語の翻訳や文章の校正を行えます。

📋

要約

「この長い文章を3行にまとめて」など、長文の要約やポイント整理が得意です。

💡

アイデア出し

「新商品のネーミング案を20個考えて」のように、ブレインストーミングのパートナーになります。

🖼️

画像の読み取り（Pro以上）

写真やスクリーンショットの内容を読み取って回答。資料の分析や画像内テキストの抽出に便利です。
