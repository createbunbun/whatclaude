💬 Obrolan Claude

Ini adalah penggunaan paling dasar. Cukup bicara dengan Claude dari browser atau aplikasi ponsel cerdas Anda dan Anda akan dapat langsung melakukan percakapan dengan AI. Anda dapat memulai secara gratis dengan membuat akun.

Platform yang didukung 🌐 Peramban (PC/Ponsel Pintar)
📱 Aplikasi iOS/Android
🖥️ Aplikasi Desktop Mac
🪟 Aplikasi desktop Windows
Rencanakan yang Anda butuhkan Tersedia dari paket gratis

Cara memulai:claude.ai Cukup kunjungi dan daftar dengan alamat email atau akun Google Anda. Yang harus Anda lakukan adalah mengetikkan pertanyaan Anda di kolom input dan Claude akan menjawabnya untuk Anda.

Anda dapat melakukan ini:

  • Membuat dan mengoreksi teks (email, laporan, artikel blog, dll.)
  • Jawaban atas pertanyaan (bantuan penelitian)
  • Diskusi/konsultasi dinding ide
  • Terjemahan/Ringkasan
  • Perhitungan dan organisasi data

📁 Proyek

Fitur ini memungkinkan Anda mengatur percakapan berdasarkan tema. Anda dapat dengan mudah menemukan komunikasi masa lalu dengan membaginya ke dalam kategori seperti "pekerjaan", "pembelajaran bahasa Inggris", dan "perencanaan perjalanan". Anda juga dapat memasukkan instruksi khusus (instruksi preset) untuk setiap proyek.

Platform yang didukung 🌐 Peramban
📱 Aplikasi ponsel pintar
Rencanakan yang Anda butuhkan Tersedia dari paket gratis

Ini seperti folder percakapan. Misalnya, jika Anda menetapkan instruksi untuk proyek ``pekerjaan'' Anda ke ``Harap gunakan bahasa yang sopan dalam email bisnis,'' Claude akan membalas dengan nada tersebut tanpa harus menentukannya setiap saat. Anda juga dapat mengunggah file dan mengajukan pertanyaan berdasarkan kontennya.

🖥️ Rekan kerja

Fitur ini memungkinkan Claude bekerja secara otomatis di belakang layar pada komputer Anda. Misalnya, Anda dapat mengatakan, "Simpan dokumen ini bersama-sama," dan Claude akan mengerjakannya di latar belakang sementara Anda mengerjakan hal lain.

Platform yang didukung 🖥️ Mac / 🪟 Aplikasi Desktop Windows
Rencanakan yang Anda butuhkan Pro dan di atasnya
⚠️ Catatan:Anda perlu menginstal aplikasi desktop (versi Mac atau Windows). Itu tidak dapat digunakan di versi browser (claude.ai) atau aplikasi ponsel pintar.

Apa itu “agen”? "Agen" di sini mengacu pada AI yang berpikir sendiri dan menjalankan tugas atas nama Anda. Obrolan normal melibatkan perjalanan bolak-balik "pertanyaan → jawaban", namun agen melanjutkan melalui beberapa langkah sambil membuat keputusan sendiri. Gambarannya adalah "asisten AI" bekerja di belakang layar.

💻 Kode Claude

Ini adalah alat AI yang berspesialisasi dalam pemrograman (pengembangan perangkat lunak). Anda dapat menyerahkan pembuatan, modifikasi, dan debugging kode Anda (menyelidiki dan memperbaiki penyebab kesalahan) kepada Claude.

Platform yang didukung 🖥️ Mac / 🪟 Windows / 🐧 Linux
(Membutuhkan instalasi di desktop)
Rencanakan yang Anda butuhkan Paket maksimal atau API bayar sesuai pemakaian
⚠️ Catatan:Diperlukan instalasi. Operasikan pada layar yang disebut "Terminal". Alat untuk pemrogram dan insinyur.

Apa itu “terminal”? Layar untuk mengeluarkan instruksi teks ke komputer. Tidak ada tombol atau menu seperti pada aplikasi yang biasa Anda gunakan, dan Anda mengoperasikannya dengan memasukkan perintah (instruksi) pada keyboard. Ini adalah ``Terminal'' di Mac, dan ``Command Prompt'' atau ``PowerShell'' di Windows.

Claude Code berjalan di terminal ini dan membaca file proyek Anda saat Anda menulis dan memodifikasi kode Anda.

⚡Keterampilan

Fungsi pintasan yang memungkinkan Anda menyimpan prosedur kerja yang sering digunakan dan memanggilnya dengan satu perintah. Ini menyelamatkan Anda dari kesulitan memasukkan instruksi yang sama berulang kali.

Platform yang didukung Gunakan dalam Kode Claude
(🖥️ Mac / 🪟 Windows / 🐧 Linux)
Rencanakan yang Anda butuhkan Sama seperti Claude Code (Paket maks atau API bayar sesuai pemakaian)

Ini seperti “resep” dalam memasak. Misalnya, jika Anda menyimpan serangkaian langkah seperti ``Jalankan pengujian dan perbaiki jika gagal'' sebagai keterampilan, Anda dapat menjalankannya secara otomatis di lain waktu dengan satu perintah singkat. Ini adalah fitur yang berguna bagi mereka yang sudah menggunakan Claude Code.

🔌 Integrasi

Sebuah fitur yang memungkinkan Claude bekerja dengan layanan dan alat lain. Muat file dari Google Drive atau gunakan hasil pencarian web untuk menjawab pertanyaan Anda.

Platform yang kompatibel (bervariasi tergantung mitra)
  • Integrasi Google Drive 🌐 Peramban
  • Pencarian web 🌐 Peramban / 📱 Aplikasi
  • Ekstensi Chrome 🌐 Hanya browser Chrome
  • Tautan Excel/PowerPoint 🪟 Windows / 🖥️ Mac
  • Integrasi kendur Di Kendur
  • MCP (Protokol Koneksi Alat Eksternal) Dalam Kode Claude (🖥️ Mac / 🪟 Windows / 🐧 Linux)
Rencanakan yang Anda butuhkan Bervariasi tergantung pada fungsinya (banyak yang Pro atau lebih tinggi)

Hal-hal yang tidak dapat dilakukan hanya dengan Claude dapat dilakukan dengan ekstensi. Misalnya, jika Anda mengaktifkan integrasi Google Drive, Anda akan dapat mengajukan pertanyaan kepada Claude tentang konten dokumen dan spreadsheet Anda di Drive.

Lihat detail ekstensi →

🔧API

Ini adalah mekanisme bagi orang-orang yang membuat aplikasi dan layanan mereka sendiri untuk memanggil fungsi Claude dari dalam program mereka.

Platform yang didukung Panggilan dari program (apa pun OSnya)
Rencanakan yang Anda butuhkan Bayar sesuai pemakaian (hanya membayar sesuai penggunaan)

"API" adalah sesuatu seperti jendela untuk bertukar data antar aplikasi. Misalnya, jika Anda ingin menyematkan "fitur obrolan AI" di situs web Anda, Anda dapat menggunakan API untuk mengirim pertanyaan ke Claude dan menerima jawaban. Pengetahuan pemrograman diperlukan, dan biaya didasarkan pada jumlah yang digunakan. Ini bukan untuk pengguna umum, tapi untuk pengembang dan insinyur.
