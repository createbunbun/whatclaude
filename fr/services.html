💬Claude Chat

C'est l'utilisation la plus basique. Parlez simplement à Claude depuis votre navigateur ou votre application pour smartphone et vous pourrez immédiatement avoir une conversation avec l'IA. Vous pouvez commencer gratuitement en créant un compte.

Plateformes prises en charge 🌐 Navigateur (PC/Smartphone)
📱Application iOS/Android
🖥️ Application de bureau Mac
🪟 Application de bureau Windows
Plan dont vous avez besoin Disponible à partir du plan gratuit

Comment commencer :claude.ai Visitez simplement et inscrivez-vous avec votre adresse e-mail ou votre compte Google. Il vous suffit de taper votre question dans le champ de saisie et Claude y répondra pour vous.

Vous pouvez faire ceci :

  • Création et correction de textes (emails, rapports, articles de blog, etc.)
  • Réponses aux questions (aide à la recherche)
  • Discussion/consultation sur le mur d’idées
  • Traduction/Résumé
  • Calcul et organisation des données

Afficher les détails du chat →

📁 Projets

Cette fonctionnalité permet d'organiser les conversations par thème. Vous pouvez facilement retrouver des communications antérieures en les divisant en catégories telles que « travail », « apprentissage de l'anglais » et « planification de voyage ». Vous pouvez également saisir des instructions personnalisées (instructions prédéfinies) pour chaque projet.

Plateformes prises en charge 🌐 Navigateur
📱 Application pour smartphone
Plan dont vous avez besoin Disponible à partir du plan gratuit

C'est comme un dossier de conversations. Par exemple, si vous définissez une instruction pour votre projet « travail » sur « Veuillez utiliser un langage poli dans les e-mails professionnels », Claude répondra sur ce ton sans avoir à le préciser à chaque fois. Vous pouvez également télécharger des fichiers et poser des questions en fonction de leur contenu.

Afficher les détails du chat/des projets →

🖥️ Cotravail

Cette fonctionnalité permet à Claude de travailler automatiquement en coulisses sur votre ordinateur. Par exemple, vous pouvez dire « Conservez ce document ensemble » et Claude travaillera dessus en arrière-plan pendant que vous travaillerez sur autre chose.

Plateformes prises en charge 🖥️ Mac / 🪟 Application de bureau Windows
Plan dont vous avez besoin Pro et supérieur
⚠️ Note:Vous devrez installer l'application de bureau (version Mac ou Windows). Il ne peut pas être utilisé dans la version navigateur (claude.ai) ni dans l'application smartphone.

Qu'est-ce qu'un « agent » ? Le « agent » fait ici référence à une IA qui pense par elle-même et effectue des tâches en votre nom. Le chat normal implique un aller-retour « question → réponse », mais les agents passent par plusieurs étapes tout en prenant leurs propres décisions. L’image est qu’un « assistant IA » travaille dans les coulisses.

Voir les détails sur Cowork →

💻Claude Code See More

Il s'agit d'un outil d'IA spécialisé dans la programmation (développement de logiciels). Vous pouvez confier la création, la modification et le débogage de votre code (recherche et correction de la cause des erreurs) à Claude.

Plateformes prises en charge 🖥️ Mac / 🪟 Windows / 🐧 Linux
(Nécessite une installation sur le bureau)
Plan dont vous avez besoin Forfait Max ou API avec paiement à l'utilisation
⚠️ Note:Installation requise. Opérez sur un écran appelé "Terminal". Un outil pour les programmeurs et les ingénieurs.

Qu'est-ce qu'un « terminal » ? Un écran pour émettre des instructions textuelles à un ordinateur. Il n'y a pas de boutons ou de menus comme dans les applications que vous utilisez habituellement, et vous l'utilisez en saisissant des commandes (instructions) sur le clavier. Il s'agit de « Terminal » sur Mac et de « Invite de commandes » ou « PowerShell » sur Windows.

Claude Code s'exécute sur ce terminal et lit les fichiers de votre projet pendant que vous écrivez et modifiez votre code.

⚡Compétences

Une fonction de raccourci qui permet de sauvegarder les procédures de travail fréquemment utilisées et de les appeler avec une seule commande. Cela vous évite d’avoir à saisir encore et encore les mêmes instructions.

Plateformes prises en charge Utilisation dans Claude Code
(🖥️ Mac / 🪟 Windows / 🐧 Linux)
Plan dont vous avez besoin Identique à Claude Code (forfait Max ou API avec paiement à l'utilisation)

C'est comme une « recette » en cuisine. Par exemple, si vous enregistrez une série d'étapes telles que « Exécuter un test et corrigez-le s'il échoue » en tant que compétence, vous pourrez l'exécuter automatiquement la prochaine fois avec une seule commande courte. C'est une fonctionnalité utile pour ceux qui utilisent déjà Claude Code.

Voir les détails des compétences →

🔌 Intégrations

Une fonctionnalité qui permet à Claude de travailler avec d'autres services et outils. Chargez des fichiers depuis Google Drive ou utilisez les résultats de recherche sur le Web pour répondre à vos questions.

Plateformes compatibles (varie selon le partenaire)
  • Intégration de Google Drive 🌐 Navigateur
  • Recherche sur le Web 🌐 Navigateur / 📱 Application
  • Extension Chrome 🌐 Navigateur Chrome uniquement
  • Lien Excel/PowerPoint 🪟 Windows / 🖥️ Mac
  • Intégration Slack Sur Slack
  • MCP (protocole de connexion d'outil externe) Dans Claude Code (🖥️ Mac / 🪟 Windows / 🐧 Linux)
Plan dont vous avez besoin Varie en fonction de la fonction (beaucoup sont Pro ou supérieurs)

Les choses que vous ne pouvez pas faire avec Claude seul peuvent être réalisées avec des extensions. Par exemple, si vous activez l'intégration de Google Drive, vous pourrez poser des questions à Claude sur le contenu de vos documents et feuilles de calcul dans Drive.

Afficher les détails de l'extension →

🔧API

Il s'agit d'un mécanisme permettant aux personnes qui créent leurs propres applications et services d'appeler les fonctions de Claude depuis leurs programmes.

Plateformes prises en charge Appel depuis le programme (quel que soit le système d'exploitation)
Plan dont vous avez besoin Pay-as-you-go (payez uniquement pour ce que vous utilisez)

"API" est quelque chose comme une fenêtre d'échange de données entre applications. Par exemple, si vous souhaitez intégrer une « fonctionnalité de chat IA » sur votre site Web, vous pouvez utiliser l'API pour envoyer des questions à Claude et recevoir des réponses. Des connaissances en programmation sont requises et les frais sont basés sur le montant utilisé. Ce n'est pas destiné aux utilisateurs généraux, mais aux développeurs et aux ingénieurs.
