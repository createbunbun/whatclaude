Comparaison des fonctionnalités

Comparez les fonctionnalités de Claude

Nous avons résumé les différences de fonctions par plan, produit et modèle dans une liste facile à comprendre.

Présentation de la fonction principale

💬
discussion textuelle

Conversation naturelle dans plusieurs langues dont le japonais et l'anglais. Nous pouvons gérer un large éventail de tâches, notamment rédiger, traduire, résumer, répondre à des questions et générer des idées.
Tous les forfaits/tous les appareils

📄
Analyse de fichiers/documents

Téléchargez des PDF, Word, Excel, des images, etc. pour lire, résumer et répondre aux questions.
Tous les forfaits

🌐
Recherche sur le Web

Recherchez les dernières informations en temps réel et reflètez-les dans vos réponses. Répondre à des informations au-delà du seuil de connaissances.
Pro et supérieur

📁
Projets

Gérez les conversations, les fichiers et les paramètres collectivement pour chaque thème. Organisez systématiquement les tâches à long terme.
Pro et supérieur

🧠
Mémoire

Une fonction de personnalisation qui mémorise les préférences et paramètres de l'utilisateur et les utilise pour les conversations futures.
Activer à partir des paramètres

💻
Génération/débogage de code

Prend en charge la création, l'explication, la révision et la correction de bogues de code dans plusieurs langages tels que Python, JavaScript et TypeScript.
Tous les forfaits

🎨
Artefacts

Une fonction qui permet de prévisualiser les pages HTML, les composants, SVG, etc. dans le chat.
Version Web / Version application

🔧
Utilisation des outils (API)

Possibilité d'appeler des fonctions personnalisées via l'API (Tool Use/Function Calling). Utilisé pour la collaboration avec des services externes.
Lors de l'utilisation de l'API
