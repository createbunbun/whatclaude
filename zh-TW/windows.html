🖥️ 如何安裝為 PWA

您可以從桌面啟動 Claude，就像它是一個應用程式一樣。

Chrome 上的 PWA 安裝說明

1

在Chrome中開啟claude.ai並登入

像往常一樣在瀏覽器中訪問 claude.ai。

2

點擊網址列中的圖標

網址列的最右側將出現「安裝」圖示（顯示器上的箭頭）。

3

點選“安裝”按鈕

出現彈出視窗時選擇“安裝”。

4

在開始功能表/工作列註冊

將其固定到開始功能表或工作列（如果需要）會更方便。

Microsoft Edge 上的 PWA 安裝說明

1

在 Edge 中開啟 claude.ai 並登入

在 Microsoft Edge 瀏覽器中存取 claude.ai。

2

點擊右上角的“...”（設定）

開啟 Edge 的三點選單。

3

“應用程式”→“將此網站安裝為應用程式”

從選單中選擇“應用程式”項目。

4

確認“安裝”

檢查應用程式名稱並點擊“安裝”即可完成。

💻 在桌面上使用 Claude Code

對於開發人員：從 Windows、Mac 和 Linux 終端機使用

先決條件/安裝步驟

1

安裝 Node.js

nodejs.org 下載並安裝 Node.js LTS 版本。

2

取得您的 Anthropic 帳戶的 API 金鑰

console.anthropic.com 頒發 API 金鑰。

3

在 PowerShell 或命令提示字元中執行

# 安裝克勞德代碼 npm install -g @anthropic-ai/claude-code # 從工作資料夾開始 cd ~/專案/我的應用程式 # 對於 Mac/Linux cd C:\Users\你的名字\Projects\my-app # 對於 Windows 克勞德
4

使用 WSL2（建議）

Windows Subsystem for Linux (WSL2) 可讓您在 Linux 環境中更舒適地執行 Claude Code。

⚠️
使用 Claude Code 需要 Anthropic API 金鑰（付費）。API指南頁面還請檢查。

🧪 桌面測試版產品

桌面整合功能目前處於測試階段

貝塔

🌐
克勞德在 Chrome 中

作為 Google Chrome 擴充功能運行的瀏覽代理程式。 你可以讓克勞德一邊工作一邊閱讀網頁。 適用於 Windows、Mac 和 Linux 上的 Chrome。

貝塔

📊
克勞德在 Excel 中

與 Microsoft Excel 的測試版整合。 Claude 支援電子表格分析、公式產生和資料組織。

貝塔

📑
克勞德在PowerPoint中

Beta 功能，支援自動產生和編輯 PowerPoint 投影片。 您可以大幅簡化演示材料的創建。

貝塔

🖥️
協同工作

用於桌面上文件管理和任務自動化的代理工具。 它甚至可以被開發人員以外的使用者使用。
