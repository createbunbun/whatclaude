💬 克勞德聊天

這是最基本的用法。 只需透過瀏覽器或智慧型手機應用程式與克勞德交談，您就可以立即與人工智慧進行對話。 您可以透過建立帳戶免費開始。

支援的平台 🌐 瀏覽器（PC/智慧型手機）
📱 iOS/Android 應用程式
🖥️ Mac 桌面應用程序
🪟 Windows 桌面應用程式
您需要的計劃 可從免費方案取得

如何開始：claude.ai 只需造訪並使用您的電子郵件地址或 Google 帳戶註冊即可。 您所要做的就是在輸入欄位中輸入您的問題，克勞德將為您解答。

你可以這樣做：

  • 建立和更正文字（電子郵件、報告、部落格文章等）
  • 問題的答案（幫助研究）
  • 創意牆討論/諮詢
  • 翻譯/摘要
  • 計算和數據組織

看聊天詳情 →

📁 項目

此功能可讓您按主題組織對話。 您可以透過將過去的通訊分為「工作」、「英語學習」和「旅行計畫」等類別來輕鬆找到。 您也可以為每個項目輸入自訂指令（預設指令）。

支援的平台 🌐 瀏覽器
📱 智慧型手機應用程式
您需要的計劃 可從免費方案取得

它就像一個對話資料夾。 例如，如果您將“工作”項目的指示設定為“請在商務電子郵件中使用禮貌用語”，克勞德就會以這種語氣回复，而不必每次都指定。 您也可以上傳文件並根據其內容提出問題。

查看聊天/專案的詳細資訊 →

🖥️ 協同辦公

此功能允許克勞德自動在您的電腦上進行幕後工作。 例如，您可以說“將此文件放在一起”，克勞德將在您處理其他事情時在後台處理它。

支援的平台 🖥️ Mac / 🪟 Windows 桌面應用程式
您需要的計劃 專業版以上
⚠️ 筆記：您需要安裝桌面應用程式（Mac 或 Windows 版本）。 它不能在瀏覽器版本（claude.ai）或智慧型手機應用程式中使用。

什麼是「代理人」？ 這裡的「代理」是指能夠獨立思考並代表你執行任務的人工智慧。 正常的聊天涉及「問題→答案」的往返，但代理商在做出自己的決定時會經歷多個步驟。 形像地說，有一個「AI助理」正在幕後工作。

💻 克勞德代碼

它是專門用於程式設計（軟體開發）的AI工具。 您可以將程式碼的建立、修改和偵錯（調查並修正錯誤原因）留給 Claude。

支援的平台 🖥️ Mac / 🪟 Windows / 🐧 Linux
（需安裝在桌面上）
您需要的計劃 最大套餐或 API 即用即付
⚠️ 筆記：需要安裝。 在稱為“終端”的螢幕上進行操作。 程式設計師和工程師的工具。

什麼是“終端”？ 用於向電腦發出文字指令的螢幕。 沒有像您通常使用的應用程式那樣的按鈕或選單，您透過在鍵盤上輸入命令（指令）來操作它。 在 Mac 上這是“終端”，在 Windows 上是“命令提示字元”或“PowerShell”。

Claude Code 在此終端機上運行，並在您編寫和修改程式碼時讀取專案檔。

⚡技能

快捷功能可讓您儲存常用的工作程序並透過單一命令呼叫它們。 這省去了您一遍又一遍輸入相同指令的麻煩。

支援的平台 在克勞德代碼中使用
（🖥️ Mac / 🪟 Windows / 🐧 Linux）
您需要的計劃 與 Claude Code 相同（Max 套餐或 API 即用即付）

這就像烹飪中的“食譜”。 例如，如果您將「執行測試並在失敗時修復它」等一系列步驟儲存為技能，則下次可以使用單一簡短命令自動執行它。 對於已經在使用 Claude Code 的人來說，這是一個有用的功能。

🔌 集成

此功能允許 Claude 與其他服務和工具一起使用。 從 Google 雲端硬碟載入文件或使用網路搜尋結果來回答您的問題。

相容平台（因合作夥伴而異）
  • 谷歌雲端硬碟集成 🌐 瀏覽器
  • 網頁搜尋 🌐 瀏覽器 / 📱 應用程式
  • Chrome 擴充功能 🌐 僅限 Chrome 瀏覽器
  • Excel/PowerPoint連動 🪟 Windows / 🖥️ Mac
  • 鬆弛集成 鬆弛時
  • MCP（外部工具連接協定） 在克勞德程式碼（🖥️ Mac / 🪟 Windows / 🐧 Linux）
您需要的計劃 因功能而異（許多是 Pro 或更高版本）

單獨使用 Claude 無法完成的事情可以透過擴充來完成。 例如，如果您啟用 Google 雲端硬碟集成，您將能夠向克勞德詢問有關雲端硬碟中的文件和電子表格內容的問題。

🔧 API

這是一種機制，供創建自己的應用程式和服務的人們從其程式中呼叫 Claude 的函數。

支援的平台 從程式呼叫（無論作業系統為何）
您需要的計劃 即用即付（僅按使用量付費）

「API」就像是應用程式之間交換資料的視窗。 例如，如果您想在網站上嵌入“AI聊天功能”，您可以使用API​​向Claude發送問題並接收答案。 需編程知識，費用依使用量而定。 它不是針對一般用戶，而是針對開發人員和工程師。
