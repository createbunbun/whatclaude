💻 克勞德代碼

克勞德·科德

致力於編碼的人工智慧代理。 它可以從終端機（命令列）運行。

🤖
自主編碼

Claude 會自動建立、編輯、刪除檔案、執行指令、測試等。 只要告訴它“製作這個應用程式”，它就會自主編寫程式碼。

🔍
程式碼庫理解

您可以在修復錯誤、重構或新增功能之前載入現有專案資料夾並了解整個程式碼。

⚡
命令執行

自動執行 npm install、git 和測試腳本等指令。 您可以按原樣自動化開發流程（需要確認）。

🌐
網路搜尋/文件參考

在搜尋最新的圖書館資訊和文件時產生代碼。 可根據最新資訊實施。

必要的環境

Node.js 18 或更高版本 npm 或 npx macOS / Linux / Windows（建議 WSL） 人為 API 金鑰

設定說明

1

安裝 Node.js

nodejs.org 從下載並安裝 LTS 版本。

2

安裝克勞德代碼

$ npm install -g @anthropic-ai/claude-code 12秒內增加248個包
3

將 API 金鑰設定為環境變數

#macOS/Linux $ 導出 ANTHROPIC_API_KEY='sk-ant-...' # Windows PowerShell > $env:ANTHROPIC_API_KEY = 'sk-ant-...'
4

在專案資料夾中啟動

$ cd ~/專案/我的應用程式 $ 克勞德 ✓ 克勞德代碼 v1.x.x 工作目錄：/Users/you/projects/my-app 我們該怎麼辦？
