技能

技能

此功能可讓您將常用任務儲存為「食譜」並一鍵呼叫。

什麼是技能？

即使是初次使用的使用者也很容易理解

類似於烹飪“食譜”的東西

不必每次都解釋相同的步驟，只需選擇一個已保存的技能，克勞德就會遵循相同的步驟。 例如，它非常適合「建立每週報告」和「固定格式的會議記錄」等重複性任務。

推薦給這些人

對於那些覺得每次輸入相同指令很乏味的人、想要在團隊內標準化工作程序的人以及想要自動化日常任務並節省時間的人來說，此功能非常適合。

支援的平台

在克勞德代碼中使用

技能可在 Claude Code 的終端環境中使用。 相容於 Mac/Windows/Linux。

ℹ️
需要安裝克勞德代碼。 此功能適合有程式設計知識的人。

您需要的計劃

與 Claude Code 屬於同一套餐

技能是作為 Claude Code 的一部分提供的功能。最大計劃（每月 100 美元或 200 美元），或 Anthropic API 即用即付取得地點： 無需額外付費。

具體技能舉例

你可以這樣使用它
📊
日常工作

「週報」技巧

以固定格式編譯每週資料。 如果事先設定資料聚合方式和輸出格式，則可以一鍵完成報表。

🔍
發展

「程式碼審查」技巧

根據您團隊的編碼規則檢查您的程式碼。 可提前定義命名約定、錯誤處理、效能等檢查點。

🌐
翻譯

「翻譯檢查」技能

比較原文和譯文並提出改進建議。 它會自動檢查術語的一致性、細微差別的準確性、可讀性等。

📝
git

「公關創造」技巧

根據提交內容自動產生拉取請求描述。 快速建立包含變更摘要、測試步驟、審核點等的 PR。

如何創造技能

只需 3 步驟即可輕鬆創建
  1. 1

    運行 /skill 命令

    在克勞德·科德的終端中 /skill 輸入命令。 技能建立精靈將啟動。

  2. 2

    描述技能名稱和步驟

    命名您的技能並描述您希望克勞德執行的步驟。 您可以用 Markdown 格式撰寫詳細的說明。

  3. 3

    下次打電話給我就好

    你創造的技能 /skillname 您可以透過名稱來稱呼它，例如 .不再每次都解釋相同的步驟。
