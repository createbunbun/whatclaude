💬 Cláudio Bate-papo

Este é o uso mais básico. Basta falar com Claude no seu navegador ou aplicativo de smartphone e você poderá conversar com a IA imediatamente. Você pode começar gratuitamente criando uma conta.

Plataformas suportadas 🌐 Navegador (PC/Smartphone)
📱 Aplicativo iOS/Android
🖥️ Aplicativo para desktop Mac
🪟 Aplicativo de desktop do Windows
Planeje o que você precisa Disponível no plano gratuito

Como começar:claude.ai Basta visitar e registrar-se com seu endereço de e-mail ou conta do Google. Tudo que você precisa fazer é digitar sua pergunta no campo de entrada e Claude responderá para você.

Você pode fazer isso:

  • Criação e correção de textos (e-mails, relatórios, artigos de blog, etc.)
  • Respostas às perguntas (ajuda na pesquisa)
  • Discussão/consulta no mural de ideias
  • Tradução/Resumo
  • Cálculos e organização de dados

📁 Projetos

Este recurso permite organizar as conversas por tema. Você pode encontrar facilmente comunicações anteriores dividindo-as em categorias como “trabalho”, “aprendizado de inglês” e “planejamento de viagens”. Você também pode inserir instruções personalizadas (instruções predefinidas) para cada projeto.

Plataformas suportadas 🌐 Navegador
📱 Aplicativo para smartphone
Planeje o que você precisa Disponível no plano gratuito

É como uma pasta de conversas. Por exemplo, se você definir uma instrução para seu projeto de “trabalho” como “Por favor, use linguagem educada em e-mails comerciais”, Claude responderá nesse tom, sem precisar especificá-lo todas as vezes. Você também pode fazer upload de arquivos e fazer perguntas com base em seu conteúdo.

Ver detalhes do bate-papo/projetos →

🖥️ Cowork

Este recurso permite que Claude trabalhe automaticamente nos bastidores do seu computador. Por exemplo, você pode dizer “Mantenha este documento sob controle” e Claude trabalhará nele em segundo plano enquanto você trabalha em outra coisa.

Plataformas suportadas 🖥️ Aplicativo para desktop Mac / 🪟 Windows
Planeje o que você precisa Pró e superior
⚠️ Observação:Você precisará instalar o aplicativo de desktop (versão Mac ou Windows). Não pode ser utilizado na versão para navegador (claude.ai) ou aplicativo para smartphone.

O que é um “agente”? O “agente” aqui se refere a uma IA que pensa por conta própria e executa tarefas em seu nome. O bate-papo normal envolve um ciclo de "pergunta → resposta", mas os agentes passam por várias etapas enquanto tomam suas próprias decisões. A imagem é que um “assistente de IA” está trabalhando nos bastidores.

Veja detalhes sobre o Cowork →

💻 Código Claude

É uma ferramenta de IA especializada em programação (desenvolvimento de software). Você pode deixar a criação, modificação e depuração do seu código (investigação e correção da causa dos erros) para Claude.

Plataformas suportadas 🖥️ Mac / 🪟 Windows / 🐧 Linux
(Requer instalação no desktop)
Planeje o que você precisa Plano máximo ou API pré-paga
⚠️ Observação:Instalação necessária. Opere em uma tela chamada “Terminal”. Uma ferramenta para programadores e engenheiros.

O que é um “terminal”? Uma tela para emitir instruções de texto para um computador. Não há botões ou menus como nos aplicativos que você costuma usar, e você o opera digitando comandos (instruções) no teclado. Este é ``Terminal'' no Mac, e ``Prompt de Comando'' ou ``PowerShell'' no Windows.

Claude Code é executado neste terminal e lê os arquivos do seu projeto enquanto você escreve e modifica seu código.

⚡Habilidades

Uma função de atalho que permite salvar procedimentos de trabalho usados ​​com frequência e acessá-los com um único comando. Isso evita o trabalho de inserir as mesmas instruções repetidamente.

Plataformas suportadas Use dentro do Código Claude
(🖥️ Mac / 🪟 Windows / 🐧 Linux)
Planeje o que você precisa Igual ao Claude Code (plano máximo ou API pré-paga)

É como uma “receita” culinária. Por exemplo, se você salvar uma série de etapas como ``Executar um teste e corrigi-lo se ele falhar'' como uma habilidade, você poderá executá-lo automaticamente na próxima vez com um único comando curto. Este é um recurso útil para quem já usa o Claude Code.

Ver detalhes das habilidades →

🔌 Integrações

Um recurso que permite ao Claude trabalhar com outros serviços e ferramentas. Carregue arquivos do Google Drive ou use os resultados de pesquisa na web para responder às suas perguntas.

Plataformas compatíveis (varia dependendo do parceiro)
  • Integração com o Google Drive 🌐 Navegador
  • Pesquisa na web 🌐 Navegador / 📱 Aplicativo
  • Extensão do Chrome 🌐 Somente navegador Chrome
  • Vinculação Excel/PowerPoint 🪟 Windows / 🖥️ Mac
  • Integração lenta No Slack
  • MCP (protocolo de conexão de ferramenta externa) No Código Claude (🖥️ Mac / 🪟 Windows / 🐧 Linux)
Planeje o que você precisa Varia dependendo da função (muitos são Pro ou superior)

Coisas que você não pode fazer sozinho com Claude podem ser feitas com extensões. Por exemplo, se você ativar a integração com o Google Drive, poderá fazer perguntas ao Claude sobre o conteúdo dos seus documentos e planilhas no Drive.

Ver detalhes da extensão →

🔧 API

Este é um mecanismo para pessoas que estão criando seus próprios aplicativos e serviços chamarem as funções de Claude de dentro de seus programas.

Plataformas suportadas Chamada do programa (independentemente do sistema operacional)
Planeje o que você precisa Pague conforme o uso (pague apenas pelo que usar)

"API" é algo como uma janela para troca de dados entre aplicativos. Por exemplo, se quiser incorporar um "recurso de chat de IA" em seu site, você pode usar a API para enviar perguntas a Claude e receber respostas. É necessário conhecimento de programação e as taxas são baseadas no valor utilizado. Não é para usuários em geral, mas para desenvolvedores e engenheiros.
