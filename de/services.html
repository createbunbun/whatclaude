💬 Claude Chat

Dies ist die grundlegendste Verwendung. Sprechen Sie einfach über Ihren Browser oder Ihre Smartphone-App mit Claude und Sie können sofort ein Gespräch mit der KI führen. Sie können kostenlos loslegen, indem Sie ein Konto erstellen.

Unterstützte Plattformen 🌐 Browser (PC/Smartphone)
📱 iOS/Android-App
🖥️ Mac-Desktop-App
🪟 Windows-Desktop-App
Planen Sie, was Sie brauchen Verfügbar im kostenlosen Plan

So fangen Sie an:claude.ai Besuchen Sie uns einfach und registrieren Sie sich mit Ihrer E-Mail-Adresse oder Ihrem Google-Konto. Sie müssen lediglich Ihre Frage in das Eingabefeld eingeben und Claude wird sie für Sie beantworten.

Sie können dies tun:

  • Erstellen und Korrigieren von Texten (E-Mails, Berichte, Blogartikel usw.)
  • Antworten auf Fragen (Hilfe bei der Recherche)
  • Diskussion/Beratung an der Ideenwand
  • Übersetzung/Zusammenfassung
  • Berechnungen und Datenorganisation

📁 Projekte

Mit dieser Funktion können Sie Gespräche nach Themen organisieren. Sie können frühere Mitteilungen leicht finden, indem Sie sie in Kategorien wie „Arbeit“, „Englisch lernen“ und „Reiseplanung“ unterteilen. Sie können für jedes Projekt auch benutzerdefinierte Anweisungen (voreingestellte Anweisungen) eingeben.

Unterstützte Plattformen 🌐 Browser
📱 Smartphone-App
Planen Sie, was Sie brauchen Verfügbar im kostenlosen Plan

Es ist wie ein Ordner mit Gesprächen. Wenn Sie beispielsweise eine Anweisung für Ihr „Arbeits“-Projekt festlegen: „Bitte verwenden Sie in geschäftlichen E-Mails eine höfliche Sprache“, antwortet Claude in diesem Ton, ohne ihn jedes Mal angeben zu müssen. Sie können auch Dateien hochladen und Fragen zu deren Inhalt stellen.

🖥️ Cowork

Mit dieser Funktion kann Claude automatisch hinter den Kulissen Ihres Computers arbeiten. Sie können beispielsweise sagen: „Behalten Sie dieses Dokument beisammen“, und Claude wird im Hintergrund daran arbeiten, während Sie an etwas anderem arbeiten.

Unterstützte Plattformen 🖥️ Mac / 🪟 Windows-Desktop-App
Planen Sie, was Sie brauchen Pro und höher
⚠️ Notiz:Sie müssen die Desktop-App (Mac- oder Windows-Version) installieren. Es kann nicht in der Browserversion (claude.ai) oder Smartphone-App verwendet werden.

Was ist ein „Agent“? Mit „Agent“ ist hier eine KI gemeint, die selbstständig denkt und in Ihrem Namen Aufgaben ausführt. Ein normaler Chat beinhaltet einen Hin- und Rückweg von „Frage → Antwort“, aber Agenten durchlaufen mehrere Schritte, während sie ihre eigenen Entscheidungen treffen. Das Bild zeigt, dass hinter den Kulissen ein „KI-Assistent“ arbeitet.

💻 Claude Code

Es handelt sich um ein auf Programmierung (Softwareentwicklung) spezialisiertes KI-Tool. Sie können Claude die Erstellung, Änderung und das Debuggen Ihres Codes (Untersuchung und Behebung der Fehlerursache) überlassen.

Unterstützte Plattformen 🖥️ Mac / 🪟 Windows / 🐧 Linux
(Erfordert Installation auf dem Desktop)
Planen Sie, was Sie brauchen Max-Plan oder API-Pay-as-you-go
⚠️ Notiz:Installation erforderlich. Arbeiten Sie auf einem Bildschirm namens „Terminal“. Ein Werkzeug für Programmierer und Ingenieure.

Was ist ein „Terminal“? Ein Bildschirm zum Ausgeben von Textanweisungen an einen Computer. Es gibt keine Tasten oder Menüs wie in den Apps, die Sie normalerweise verwenden, und Sie bedienen es, indem Sie Befehle (Anweisungen) auf der Tastatur eingeben. Dies ist „Terminal“ auf dem Mac und „Eingabeaufforderung“ oder „PowerShell“ unter Windows.

Claude Code wird auf diesem Terminal ausgeführt und liest die Dateien Ihres Projekts, während Sie Ihren Code schreiben und ändern.

⚡Fähigkeiten

Eine Shortcut-Funktion, mit der Sie häufig verwendete Arbeitsabläufe speichern und mit einem einzigen Befehl aufrufen können. Dies erspart Ihnen die Mühe, immer wieder dieselben Anweisungen einzugeben.

Unterstützte Plattformen Verwendung innerhalb von Claude Code
(🖥️ Mac / 🪟 Windows / 🐧 Linux)
Planen Sie, was Sie brauchen Wie Claude Code (Max-Plan oder API-Pay-as-you-go)

Es ist wie ein „Rezept“ beim Kochen. Wenn Sie beispielsweise eine Reihe von Schritten wie „Führen Sie einen Test durch und beheben Sie ihn, wenn er fehlschlägt“ als Skill speichern, können Sie ihn beim nächsten Mal mit einem einzigen kurzen Befehl automatisch ausführen. Dies ist eine nützliche Funktion für diejenigen, die Claude Code bereits verwenden.

🔌 Integrationen

Eine Funktion, die es Claude ermöglicht, mit anderen Diensten und Tools zu arbeiten. Laden Sie Dateien von Google Drive oder verwenden Sie Websuchergebnisse, um Ihre Fragen zu beantworten.

Kompatible Plattformen (variiert je nach Partner)
  • Google Drive-Integration 🌐 Browser
  • Websuche 🌐 Browser / 📱 App
  • Chrome-Erweiterung 🌐 Nur Chrome-Browser
  • Excel/PowerPoint-Verknüpfung 🪟 Windows / 🖥️ Mac
  • Slack-Integration Auf Slack
  • MCP (External Tool Connection Protocol) In Claude Code (🖥️ Mac / 🪟 Windows / 🐧 Linux)
Planen Sie, was Sie brauchen Variiert je nach Funktion (viele sind Pro oder höher)

Dinge, die Sie mit Claude allein nicht tun können, können Sie mit Erweiterungen erledigen. Wenn Sie beispielsweise die Google Drive-Integration aktivieren, können Sie Claude Fragen zum Inhalt Ihrer Dokumente und Tabellen in Drive stellen.

🔧API

Dies ist ein Mechanismus für Leute, die ihre eigenen Apps und Dienste erstellen, um Claudes Funktionen aus ihren Programmen heraus aufzurufen.

Unterstützte Plattformen Aufruf aus dem Programm (unabhängig vom Betriebssystem)
Planen Sie, was Sie brauchen Pay-as-you-go (zahlen Sie nur für das, was Sie nutzen)

„API“ ist so etwas wie ein Fenster zum Datenaustausch zwischen Anwendungen. Wenn Sie beispielsweise eine „KI-Chat-Funktion“ auf Ihrer Website einbetten möchten, können Sie über die API Fragen an Claude senden und Antworten erhalten. Programmierkenntnisse sind erforderlich, die Gebühren richten sich nach dem eingesetzten Betrag. Es richtet sich nicht an allgemeine Benutzer, sondern an Entwickler und Ingenieure.
