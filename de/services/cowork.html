Cowork

Cowork

Claude arbeitet im Hintergrund, während Sie an anderen Aufgaben arbeiten

Was ist Cowork?

Einfache Erklärung für Erstbenutzer

Unterschiede zum normalen Chat

Ein normaler Chat wird mit einem Rundlauf von „Frage → Antwort“ abgeschlossen.
Cowork ist anders. Sagen Sie Claude einfach, dass er dies tun soll, und Claude führt die Schritte hinter den Kulissen automatisch für Sie aus.

Als Bild,Ein KI-Assistent arbeitet still und leise hinter den Kulissen.Es ist ein Gefühl. Während Sie an anderen Aufgaben arbeiten oder eine Tasse Kaffee trinken, erledigt Claude die Aufgaben, die Sie von ihm verlangen.

Unterstützte Plattformen

  • 🖥️ Mac-Desktop-App ○ Kompatibel
  • 🪟 Windows-Desktop-App ○ Kompatibel
  • 🌐 Browserversion (claude.ai) ✕ Kann nicht verwendet werden
  • 📱 Smartphone-App (iOS/Android) ✕ Kann nicht verwendet werden
💡
Erfordert die Installation einer Desktop-App. Die Cowork-Funktion ist in der Browserversion oder Smartphone-App nicht verfügbar. Wenn Sie es noch nicht haben, laden Sie es bitte herunter, indem Sie unten auf „Erste Schritte“ klicken.

So starten Sie (3 Schritte)

1

Laden Sie die Desktop-App herunter

claude.ai/download Gehen Sie zu und laden Sie die App für Ihr Betriebssystem (Mac oder Windows) herunter.

2

Öffnen Sie die App und melden Sie sich an

Starten Sie die heruntergeladene App und melden Sie sich mit Ihrem Claude-Konto an. Stellen Sie sicher, dass Sie einen Pro-Plan oder höher abonniert haben.

3

Modus „Cowork“ auswählen und anweisen

Wählen Sie auf dem Chat-Bildschirm den Modus „Cowork“ und weisen Sie die Arbeit an, die Sie erledigen möchten. Den Rest erledigt Claude automatisch im Hintergrund.
