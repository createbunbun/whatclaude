빠른 시작

Python SDK를 사용한 기본 예

1

SDK 설치

pip install anthropic
2

API 키 설정

수출 ANTHROPIC_API_KEY='your-api-key-here'
3

메시지 보내기

import anthropic client = anthropic.Anthropic() message = client.messages.create( model="claude-sonnet-4-6", max_tokens=1024년, messages=[ {"롤": "user", "content": "안녕하세요!"} ] ) 인쇄(message.content)

Node.js / TypeScript SDK를 사용한 기본 예

1

SDK 설치

npm install @anthropic-ai/sdk
2

코드 작성

import Anthropic from '@anthropic-ai/sdk'; const client = 새로운 Anthropic ({ apiKey: process.env.ANTHROPIC_API_KEY, }); const message= await client.messages.create({ model: 'claude-sonnet-4-6', max_tokens: 1024년, messages: [ { role: 'user', content: '일본어로 안녕하세요! ' } ], }); console.log(message.content);

curl의 요청 예

curl https://api.anthropic.com/v1/messages \ --header "x-api-key: $ANTHROPIC_API_KEY" \ --header "anthropic-version: 2023-06-01" \ --header "content-type: application/json" \ --data'{ "model": "claude-sonnet-4-6", "max_tokens": 1024년, "messages": [ { "롤": "user", "content": "안녕하세요!" } ] } '

주요 API 기능

💬
Messages API

기본 채팅 형식의 메시지를 보내고 받는 API입니다. 시스템 프롬프트 멀티턴 대화에 대응.

⚡
스트리밍

응답을 실시간으로 스트림 수신. 사용자 경험 향상을 위해 권장되는 구현 방법입니다.

🔧
도구 사용(Tool Use)

Claude가 외부 도구 및 함수를 호출 할 수있는 기능 (Function Calling). 데이터베이스 검색·API 제휴 등에 활용.

👁️
비전(이미지 입력)

화상을 입력으로서 건네주어, 내용을 이해·해석시키는 기능. PNG, JPEG, GIF, WebP 지원.

📦
배치 처리

대량의 요청을 비동기적으로 처리하는 Batch API. 비용 절감과 높은 처리량을 실현.

💾
캐시(Prompt Caching)

동일한 접두사의 토큰을 캐시하여 비용과 지연을 줄이는 기능. 긴 시스템 프롬프트에 효과적.

🔐 인증 및 보안

API 키 관리

  • 🔑 console.anthropic.com 에서 발행·관리
  • 🔒 키는 환경 변수 (.env)에 저장하고 코드에 하드 코딩하지 않습니다.
  • 🚫 Git 리포지토리에 키를 커밋하지 않음
  • 🔄 정기적 인 키 로테이션 권장

속도 제한

  • 📊 요청 및 토큰 수 모두 제한됨
  • ⏱️ 오류가 발생하면 지수 백 오프로 재 시도
  • 📈 이용량에 따라 비율 상한 인상 신청 가능
