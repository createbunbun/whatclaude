Skills(스킬)

자주 사용하는 작업을 "레시피"로 저장하고 한 번의 클릭으로 호출 할 수있는 기능입니다

Skills란?

처음으로도 이해하기 쉽습니다.

요리의 "레시피"와 같은 것

매번 같은 절차를 설명하지 않아도 저장된 스킬을 선택하는 것만으로 Claude가 정해진 순서로 작업해 줍니다. 예를 들면 「매주 하는 리포트 작성」 「정해진 포맷의 의사록」 등, 반복하는 작업에 최적입니다.

이런 분들에게 추천

매번 같은 지시를 입력하는 것이 번거롭게 느끼는 분, 팀 내에서 작업 순서를 통일하고 싶은 분, 정형 작업을 자동화하여 시간을 절약하고 싶은 분에게 딱 맞는 기능입니다.

지원 플랫폼

Claude Code 내에서 사용

Skills는 Claude Code 터미널 환경에서 사용할 수 있습니다. Mac / Windows / Linux 모두 지원됩니다.

ℹ️
Claude Code를 설치해야 합니다. 프로그래밍 지식이 있는 지향적인 기능입니다.

필요한 계획

Claude Code와 동일한 플랜에서 사용 가능

Skills는 Claude Code의 일부로 제공되는 기능입니다.Max 플랜(월 $100 또는 $200) 또는 Anthropic API 종량 과금에서 사용할 수 있습니다. 추가 요금은 부과되지 않습니다.

스킬의 구체적인 예

이런 사용법을 할 수 있습니다
📊
정형 업무

'주간 보고서' 기술

매주 데이터를 정해진 포맷으로 정리합니다. 데이터의 집계 방법이나 출력 형식을 미리 설정해 두면 원 클릭으로 리포트가 완성됩니다.

🔍
개발

'코드 리뷰' 기술

팀의 코딩 규칙에 따라 코드를 확인합니다. 명명 규칙, 오류 처리 및 성능과 같은 확인 포인트를 미리 정의할 수 있습니다.

🌐
번역

"번역 체크" 스킬

원문과 번역문을 비교하여 개선점을 제안합니다. 용어의 통일성, 뉘앙스의 정확성, 가독성 등을 자동으로 체크해 줍니다.

📝
Git

「PR작성」 스킬

커밋 내용에서 끌어오기 요청 설명을 자동으로 생성합니다. 변경 사항 요약, 테스트 절차, 검토 지점 등을 포함한 PR을 신속하게 만들 수 있습니다.

스킬을 만드는 법

3단계로 쉽게 제작
  1. 1

    /skill 명령 실행

    Claude Code 터미널에서 /skill 명령을 입력합니다. 스킬 생성 마법사가 시작됩니다.

  2. 2

    기술의 이름과 절차 작성

    스킬의 이름을 지정하고 Claude에서 실행해야 하는 단계를 설명합니다. Markdown 형식으로 자세한 지침을 작성할 수 있습니다.

  3. 3

    다음 번부터 불러

    만든 스킬은 /skillname 같은 이름으로 호출할 수 있습니다. 매번 동일한 절차를 설명할 필요가 없습니다.
