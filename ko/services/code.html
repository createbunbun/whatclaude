💻 Claude Code

Claude Code

코딩 전용 AI 에이전트. 터미널(명령줄)에서 작동합니다.

🤖
자율적인 코딩

파일 작성, 편집, 삭제, 명령 실행, 테스트 등은 Claude가 자동으로 수행합니다. "이 앱을 만들고"라고 지시하면 코드를 자율적으로 작성할 수 있습니다.

🔍
코드베이스 이해

기존 프로젝트 폴더를 읽고 전체 코드를 이해하고 버그 수정, 리팩토링 및 기능을 추가 할 수 있습니다.

⚡
명령 실행

npm install · git · 테스트 스크립트 등의 명령을 자동 실행. 개발 흐름을 그대로 자동화할 수 있습니다(요점 확인).

🌐
웹 검색 및 문서 참조

최신 라이브러리 정보와 문서를 검색하면서 코드를 생성합니다. 최신 정보를 바탕으로 한 구현이 가능합니다.

🚀 설치 설정

⚠️
Claude Code를 사용하려면 Anthropic API 키(유료)가 필요합니다.API 가이드 에서 키를 얻는 방법을 확인하십시오.

필요한 환경

Node.js 18 이상 npm 또는 npx macOS / Linux / Windows (WSL 권장) Anthropic API 키

설정 절차

1

Node.js 설치

nodejs.org 에서 LTS 버전을 다운로드하여 설치합니다.

2

Claude Code 설치

$ npm install -g @anthropic-ai/claude-code added 248 packages in 12s
3

API 키를 환경 변수로 설정

#macOS/Linux $ export ANTHROPIC_API_KEY='sk-ant-...' # Windows PowerShell > $env:ANTHROPIC_API_KEY = 'sk-ant-...'
4

프로젝트 폴더에서 시작

$ cd ~/projects/my-app $ claude ✓ Claude Code v1.x.x 작업 디렉토리: /Users/you/projects/my-app 무엇을 할까요?

💡 사용법의 예

새 앱 만들기

당신: React에서 일기 예보 앱을 만들어보세요. OpenWeather API를 사용하여 현재 위치의 날씨를 표시하는 간단한 것입니다. Claude: 알겠습니다. 일기 예보 React 앱을 만듭니다. ✓ 프로젝트 구조 확인 중 ... ✓ src/App.jsx 생성 중... ✓ src/components/WeatherCard.jsx 생성 중... ✓ src/hooks/useWeather.js 작성 중 ... ✓ package.json 업데이트 중 ... ✓ 완료! `npm start`로 시작할 수 있습니다.

버그 수정

당신: 로그인시 `TypeError: Cannot read property 'id' of undefined` 가 나온다. 고쳐. Claude: 오류를 조사합니다. ✓ src/auth/login.js 로딩 중 ... ✓ 문제 발견: user 객체의 null 검사 부족 ✓ src/auth/login.js 수정 중 ... ✓ 수정 완료! 변경 내용: - 수정 전: const userId = user.id + 수정 후: const userId = user?.id ?? null

리팩토링

당신: utils.js의 함수가 중복되었습니다. 정리해. Claude: 코드를 구문 분석합니다. ✓ utils.js 구문 분석 중 (384 줄) ... ✓ 중복 발견 : formatDate가 3 개소에서 중복 정의 ✓ 공통 함수에 통합 중 ... ✓ 리팩토링 완료: 384행 → 201행(47% 절감)

테스트 작성

당신: src/api/userService.js의 모든 함수에 Jest 테스트를 작성하십시오. Claude: 테스트를 만듭니다. ✓ userService.js 구문 분석 중 ... ✓ 7가지 함수 감지 ✓ tests/api/userService.test.js 작성 중 ... ✓ 23개의 테스트 케이스 생성 ✓ 모의 설정 중 ... ✓ 완료! `npm test`로 실행할 수 있습니다.
