🖥️ PWA로 설치하는 방법

앱 감각으로 데스크톱에서 Claude를 시작할 수 있습니다.

Chrome에서 PWA 설치 지침

1

Chrome에서 claude.ai를 열고 로그인

평소처럼 브라우저에서 claude.ai에 액세스하십시오.

2

주소 표시줄 아이콘을 클릭합니다.

주소 표시줄의 오른쪽 끝에 설치 아이콘(모니터에 화살표)이 표시됩니다.

3

설치 버튼을 클릭

팝업이 표시되면 설치를 선택합니다.

4

시작 메뉴 작업 표시줄에 등록

시작 메뉴와 필요에 따라 작업 표시 줄에 고정하면 더 편리합니다.

Microsoft Edge에서 PWA 설치 절차

1

Edge에서 claude.ai를 열고 로그인

Microsoft Edge 브라우저에서 claude.ai에 액세스합니다.

2

오른쪽 상단의 "..."(설정)을 클릭합니다.

Edge의 세 점 메뉴를 엽니다.

3

「앱」→「이 사이트를 앱으로 설치」

메뉴에서 앱 항목을 선택합니다.

4

"설치" 확인

앱 이름을 확인하고 "설치"를 클릭하면 완료됩니다.

💻 바탕 화면에서 Claude Code 사용

개발자용: Windows, Mac, Linux 터미널에서 사용

전제 조건 및 설치 절차

1

Node.js 설치

nodejs.org 에서 Node.js LTS 버전을 다운로드하여 설치.

2

Anthropic 계정의 API 키 가져오기

console.anthropic.com 에서 API 키를 발행합니다.

3

PowerShell 또는 명령 프롬프트에서 실행

# Claude Code 설치 npm install -g @anthropic-ai/claude-code # 작업 폴더로 시작 cd ~/Projects/my-app # Mac/Linux의 경우 cd C:\Users\YourName\Projects\my-app # Windows의 경우 claude
4

WSL2 활용(권장)

Windows Subsystem for Linux (WSL2)를 사용하면 Linux 환경에서 Claude Code를보다 편안하게 실행할 수 있습니다.

⚠️
Claude Code를 사용하려면 Anthropic API 키(유료)가 필요합니다.API 가이드 페이지또한 확인하십시오.

🧪 데스크톱용 베타 제품

현재 베타로 제공 중인 데스크톱 연동 기능

베타

🌐
Claude in Chrome

Google 크롬 확장 프로그램으로 작동하는 브라우징 에이전트 웹 페이지를 읽으면서 Claude에서 작업할 수 있습니다. Windows, Mac, Linux Chrome에서 사용할 수 있습니다.

베타

📊
Claude in Excel

Microsoft Excel과의 통합 베타. 스프레드시트 분석, 수식 생성 및 데이터 정리를 Claude에서 지원합니다.

베타

📑
Claude in PowerPoint

PowerPoint 슬라이드의 자동 생성 및 편집을 지원하는 베타 기능. 프레젠테이션 자료 작성을 대폭 효율화할 수 있습니다.

베타

🖥️
Cowork

데스크탑에서 파일 관리 및 태스크 자동화를 수행하는 에이전트 도구. 개발자 이외의 사용자도 활용할 수 있습니다.

🔬
베타 기능은 변경·제공 종료될 가능성이 있습니다. 최신 정보는 claude.ai 설정 화면에서 확인하십시오.
