💬 Claude 채팅

가장 기본적인 사용법입니다. 브라우저나 스마트폰 앱에서 Claude에게 말하기만 하면 바로 AI와 대화할 수 있습니다. 계정을 만들면 무료로 시작할 수 있습니다.

지원 플랫폼 🌐 브라우저(PC·스마트폰)
📱 iOS · Android 앱
🖥️ Mac 데스크톱 앱
🪟 Windows 데스크톱 앱
필요한 계획 무료 플랜에서 이용 가능

시작하는 방법:claude.ai 로 이동하여 이메일 주소나 Google 계정으로 가입하세요. 그리고는 입력란에 질문을 넣으면 Claude가 대답해 줍니다.

이렇게 할 수 있습니다 :

  • 문장의 작성·첨삭(메일, 리포트, 블로그 기사 등)
  • 질문에 대한 답변(조사물의 도움)
  • 아이디어의 벽 치기·상담
  • 번역·요약
  • 계산 및 데이터 정리

📁 Projects(프로젝트)

대화를 테마별로 정리할 수 있는 기능입니다. '일용', '영어 학습', '여행 계획'과 같이 나누면 과거의 상호작용을 바로 찾을 수 있습니다. 프로젝트별로 맞춤설정(미리 설정해 둔 지시문)을 넣을 수도 있습니다.

지원 플랫폼 🌐 브라우저
📱 스마트 폰 앱
필요한 계획 무료 플랜에서 이용 가능

대화 폴더 같은 것입니다. 예를 들어, "작업"프로젝트에 "비즈니스 메일 경건을 사용하십시오"라는 지시를 설정하면 매번 지정하지 않아도 Claude가 그 어조로 반환합니다. 파일을 업로드하고 내용을 기반으로 하는 질문도 할 수 있습니다.

채팅 및 프로젝트에 대해 자세히 알아보기 →

🖥️ Cowork (코워크)

PC의 뒷면에서 Claude가 자동으로 작업을 진행시켜주는 기능입니다. 예를 들어, "이 문서를 함께 넣어"라고 말하면 Claude는 다른 작업을 수행하는 동안 백그라운드에서 작업합니다.

지원 플랫폼 🖥️ Mac / 🪟 Windows 데스크톱 앱
필요한 계획 Pro 이상
⚠️ 주의:데스크톱 앱(Mac 또는 Windows)을 설치해야 합니다. 브라우저 버전 (claude.ai)이나 스마트 폰 앱에서는 사용할 수 없습니다.

「에이전트」란? 여기서 말하는 「에이전트」란, 당신 대신에 스스로 생각해 작업을 진행시켜 주는 AI 입니다. 보통의 채팅은 「질문→답변」의 일왕복입니다만, 에이전트는 복수의 스텝을 스스로 판단하면서 진행해 줍니다. 이른바 "AI 어시스턴트"가 뒤에서 일하고 있는 이미지입니다.

💻 Claude Code

프로그래밍(소프트웨어 개발)에 특화된 AI 툴입니다. 코드 작성, 수정, 디버깅 (오류의 원인 조사 및 수정)을 Claude에 맡길 수 있습니다.

지원 플랫폼 🖥️ Mac / 🪟 Windows / 🐧 Linux
(데스크톱에 설치 필요)
필요한 계획 Max 플랜 또는 API 종량 과금
⚠️ 주의:설치가 필요합니다. "터미널"이라는 화면에서 조작합니다. 프로그래머 엔지니어를 위한 도구입니다.

「터미널」이란? PC에 문자로 지시를 내기 위한 화면입니다. 평소 사용하는 앱처럼 버튼이나 메뉴는 없고 키보드로 명령(명령문)을 입력하여 조작합니다. Mac에서는 "터미널", Windows에서는 "명령 프롬프트"와 "PowerShell"이 있습니다.

Claude Code는 이 터미널에서 움직이며 프로젝트의 파일을 읽으면서 코드를 다시 작성하거나 다시 작성합니다.

⚡ Skills (스킬)

자주 사용하는 작업 순서를 저장해 두고, 원 커맨드로 호출할 수 있는 쇼트 컷 기능입니다. 여러 번 같은 지시를 치는 번거로움을 줄일 수 있습니다.

지원 플랫폼 Claude Code 내에서 사용
(🖥️ Mac / 🪟 Windows / 🐧 Linux)
필요한 계획 Claude Code와 동일(Max 플랜 또는 API 종량 청구)

요리에서 말하는 「레시피」와 같은 것입니다. 예를 들어 "테스트를 실행하고 실패하면 수정한다"라는 일련의 단계를 스킬로 저장해 두면 다음부터는 짧은 명령 하나로 자동 실행할 수 있습니다. Claude Code를 이미 사용하고 있는 편의 편리한 기능입니다.

🔌 확장(Integrations)

Claude를 다른 서비스 및 도구와 연결하는 기능입니다. Google 드라이브 파일을 로드하거나 웹 검색 결과를 사용하여 답변할 수 있습니다.

대응 플랫폼(연계처마다 다릅니다)
  • Google 드라이브 연동 🌐 브라우저
  • 웹 검색 🌐 브라우저 / 📱 앱
  • Chrome 확장 프로그램 🌐 Chrome 브라우저 전용
  • Excel/PowerPoint 연동 🪟 Windows / 🖥️ Mac
  • Slack 협력 Slack 상단
  • MCP(외부 공구 연결 프로토콜) Claude Code 내(🖥️ Mac / 🪟 Windows / 🐧 Linux)
필요한 계획 기능에 따라 다름(많은 프로 이상)

Claude 단독으로는 할 수 없는 것도, 확장 기능을 사용하면 가능하게 됩니다. 예를 들어 Google 드라이브 연동을 켜면 드라이브의 문서와 스프레드 시트 내용에 대해 Claude에 질문 할 수 있습니다.

🔧 API

스스로 앱이나 서비스를 만들고 있는 사람이, 그 프로그램 중에서 Claude의 기능을 호출하기 위한 구조입니다.

지원 플랫폼 프로그램에서 호출(OS 불문)
필요한 계획 종량 과금(사용한 분만 지불)

"API"는 애플리케이션이 데이터를 주고 받는 창구와 같습니다. 예를 들어 웹 사이트에 AI 채팅 기능을 포함하려는 경우 API를 사용하여 Claude에 질문을 보내고 답변을 받을 수 있습니다. 프로그래밍 지식이 필요하며 요금은 사용한 양에 따라 발생합니다. 일반 사용자용이 아닌 개발자 엔지니어용입니다.
