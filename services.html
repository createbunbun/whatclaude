Claude のサービス一覧

どのサービスが自分に合うか、ひと目でわかるガイド。
プラン・対応環境ごとに整理しました。

💬 Claude チャット

もっとも基本的な使い方です。ブラウザやスマホアプリから Claude に話しかけるだけで、すぐに AI と会話できます。アカウントを作れば無料で始められます。

対応プラットフォーム 🌐 ブラウザ（PC・スマホ）
📱 iOS・Android アプリ
🖥️ Mac デスクトップアプリ
🪟 Windows デスクトップアプリ
必要なプラン 無料プランから利用可能

はじめ方：claude.ai にアクセスして、メールアドレスか Google アカウントで登録するだけ。あとは入力欄に質問を打ち込めば Claude が答えてくれます。

こんなことができます：

  • 文章の作成・添削（メール、レポート、ブログ記事など）
  • 質問への回答（調べものの手助け）
  • アイデアの壁打ち・相談
  • 翻訳・要約
  • 計算やデータの整理

📁 Projects（プロジェクト）

会話をテーマごとに整理できる機能です。「仕事用」「英語学習」「旅行計画」のように分けておくと、過去のやり取りがすぐに見つかります。プロジェクトごとにカスタム指示（あらかじめ設定しておく指示文）を入れることもできます。

対応プラットフォーム 🌐 ブラウザ
📱 スマホアプリ
必要なプラン 無料プランから利用可能

会話のフォルダのようなものです。たとえば「仕事」プロジェクトには「ビジネスメールの敬語を使ってください」という指示を設定しておけば、毎回指定しなくても Claude がその口調で返してくれます。ファイルをアップロードして、その内容に基づいた質問もできます。

チャット・Projects の詳細を見る →

🖥️ Cowork（コワーク）

パソコンの裏側で Claude が自動的に作業を進めてくれる機能です。たとえば「この資料をまとめておいて」と指示すると、あなたが別の作業をしている間に Claude がバックグラウンドで作業してくれます。

対応プラットフォーム 🖥️ Mac / 🪟 Windows デスクトップアプリ
必要なプラン Pro 以上
⚠️ 注意：デスクトップアプリ（Mac 版または Windows 版）をインストールする必要があります。ブラウザ版（claude.ai）やスマホアプリでは使えません。

「エージェント」とは？ ここでいう「エージェント」とは、あなたの代わりに自分で考えて作業を進めてくれる AI のことです。普通のチャットは「質問→回答」の一往復ですが、エージェントは複数のステップを自分で判断しながら進めてくれます。いわば "AI アシスタント" が裏で働いているイメージです。

💻 Claude Code

プログラミング（ソフトウェア開発）に特化した AI ツールです。コードの作成・修正・デバッグ（エラーの原因調査と修正）を Claude に任せることができます。

対応プラットフォーム 🖥️ Mac / 🪟 Windows / 🐧 Linux
（デスクトップにインストールが必要）
必要なプラン Max プラン、または API 従量課金
⚠️ 注意：インストールが必要です。「ターミナル」と呼ばれる画面で操作します。プログラマー・エンジニア向けのツールです。

「ターミナル」とは？ パソコンに文字で指示を出すための画面のことです。普段使うアプリのようにボタンやメニューはなく、キーボードでコマンド（命令文）を入力して操作します。Mac では「ターミナル」、Windows では「コマンドプロンプト」や「PowerShell」がこれにあたります。

Claude Code はこのターミナル上で動き、プロジェクトのファイルを読み取りながらコードを書いたり直したりしてくれます。

⚡ Skills（スキル）

よく使う作業手順を保存しておき、ワンコマンドで呼び出せるショートカット機能です。何度も同じ指示を打ち込む手間を省けます。

対応プラットフォーム Claude Code 内で利用
（🖥️ Mac / 🪟 Windows / 🐧 Linux）
必要なプラン Claude Code と同じ（Max プランまたは API 従量課金）

料理でいう「レシピ」のようなものです。たとえば「テストを実行して、失敗したら修正する」という一連の手順をスキルとして保存しておけば、次からは短いコマンドひとつで自動実行できます。Claude Code をすでに使っている方向けの便利機能です。

🔌 拡張機能（Integrations）

Claude を他のサービスやツールと連携させる機能です。Google ドライブのファイルを読み込ませたり、Web 検索の結果を使って回答させたりできます。

対応プラットフォーム（連携先ごとに異なります）
  • Google ドライブ連携 🌐 ブラウザ
  • Web 検索 🌐 ブラウザ / 📱 アプリ
  • Chrome 拡張機能 🌐 Chrome ブラウザのみ
  • Excel / PowerPoint 連携 🪟 Windows / 🖥️ Mac
  • Slack 連携 Slack 上
  • MCP（外部ツール接続プロトコル） Claude Code 内（🖥️ Mac / 🪟 Windows / 🐧 Linux）
必要なプラン 機能によって異なる（多くは Pro 以上）

Claude 単体ではできないことも、拡張機能を使えば可能になります。たとえば Google ドライブ連携をオンにすると、ドライブ内のドキュメントやスプレッドシートの内容について Claude に質問できるようになります。

拡張機能の詳細を見る →

🔧 API

自分でアプリやサービスを作っている人が、そのプログラムの中から Claude の機能を呼び出すための仕組みです。

対応プラットフォーム プログラムから呼び出し（OS 不問）
必要なプラン 従量課金（使った分だけ支払い）

「API」とは、アプリケーションどうしがデータをやり取りするための窓口のようなものです。たとえば自分の Web サイトに「AI チャット機能」を埋め込みたいとき、API を使って Claude に質問を送り、回答を受け取ることができます。プログラミングの知識が必要で、料金は使った量に応じて発生します。一般ユーザー向けではなく、開発者・エンジニア向けです。
