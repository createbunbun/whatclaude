💬 Chat de Claude

Este es el uso más básico. Simplemente habla con Claude desde tu navegador o aplicación de teléfono inteligente y podrás tener una conversación con la IA de inmediato. Puede comenzar de forma gratuita creando una cuenta.

Plataformas compatibles 🌐 Navegador (PC/teléfono inteligente)
📱 Aplicación para iOS/Android
🖥️ Aplicación de escritorio para Mac
🪟 Aplicación de escritorio de Windows
Planifica lo que necesitas Disponible desde el plan gratuito

Cómo empezar:claude.ai Simplemente visite y regístrese con su dirección de correo electrónico o cuenta de Google. Todo lo que tienes que hacer es escribir tu pregunta en el campo de entrada y Claude la responderá por ti.

Puedes hacer esto:

  • Creación y corrección de textos (correos electrónicos, informes, artículos de blog, etc.)
  • Respuestas a preguntas (ayuda con la investigación)
  • Discusión/consulta en el muro de ideas
  • Traducción/Resumen
  • Cálculo y organización de datos.

📁 Proyectos

Esta función le permite organizar conversaciones por tema. Puede encontrar fácilmente comunicaciones anteriores dividiéndolas en categorías como "trabajo", "aprendizaje de inglés" y "planificación de viajes". También puede ingresar instrucciones personalizadas (instrucciones preestablecidas) para cada proyecto.

Plataformas compatibles 🌐 Navegador
📱 Aplicación para teléfono inteligente
Planifica lo que necesitas Disponible desde el plan gratuito

Es como una carpeta de conversaciones. Por ejemplo, si configura una instrucción para su proyecto de "trabajo" como "Utilice un lenguaje cortés en los correos electrónicos comerciales", Claude responderá en ese tono sin tener que especificarlo cada vez. También puedes subir archivos y hacer preguntas basadas en su contenido.

🖥️Cotrabajo

Esta característica le permite a Claude trabajar automáticamente detrás de escena en su computadora. Por ejemplo, puedes decir "Mantén este documento junto" y Claude trabajará en él en segundo plano mientras tú trabajas en otra cosa.

Plataformas compatibles 🖥️ Mac / 🪟 Aplicación de escritorio de Windows
Planifica lo que necesitas Pro y superior
⚠️ Nota:Deberá instalar la aplicación de escritorio (versión para Mac o Windows). No se puede utilizar en la versión del navegador (claude.ai) ni en la aplicación para teléfonos inteligentes.

¿Qué es un “agente”? El "agente" aquí se refiere a una IA que piensa por sí misma y lleva a cabo tareas en su nombre. El chat normal implica un viaje de ida y vuelta de "pregunta → respuesta", pero los agentes siguen varios pasos mientras toman sus propias decisiones. La imagen es que un "asistente de IA" está trabajando detrás de escena.

💻 Código Claude

Es una herramienta de IA especializada en programación (desarrollo de software). Puede dejar la creación, modificación y depuración de su código (investigando y corrigiendo la causa de los errores) a Claude.

Plataformas compatibles 🖥️ Mac / 🪟 Windows / 🐧 Linux
(Requiere instalación en escritorio)
Planifica lo que necesitas Plan máximo o API de pago por uso
⚠️ Nota:Requiere instalación. Operar en una pantalla llamada "Terminal". Una herramienta para programadores e ingenieros.

¿Qué es una “terminal”? Una pantalla para enviar instrucciones de texto a una computadora. No hay botones ni menús como en las aplicaciones que usas habitualmente, y puedes operarlo ingresando comandos (instrucciones) en el teclado. Esto es "Terminal" en Mac y "Símbolo del sistema" o "PowerShell" en Windows.

Claude Code se ejecuta en esta terminal y lee los archivos de su proyecto a medida que escribe y modifica su código.

⚡Habilidades

Una función de acceso directo que le permite guardar procedimientos de trabajo utilizados con frecuencia y recuperarlos con un solo comando. Esto le ahorra la molestia de ingresar las mismas instrucciones una y otra vez.

Plataformas compatibles Usar dentro del Código Claude
(🖥️Mac/🪟Windows/🐧Linux)
Planifica lo que necesitas Igual que Claude Code (plan Max o API de pago por uso)

Es como una "receta" de cocina. Por ejemplo, si guarda una serie de pasos como "Ejecutar una prueba y corregirla si falla" como una habilidad, podrá ejecutarla automáticamente la próxima vez con un único comando corto. Esta es una característica útil para aquellos que ya utilizan Claude Code.

🔌 Integraciones

Una característica que le permite a Claude trabajar con otros servicios y herramientas. Cargue archivos desde Google Drive o utilice los resultados de búsqueda web para responder sus preguntas.

Plataformas compatibles (varía según el socio)
  • Integración con Google Drive 🌐 Navegador
  • búsqueda web 🌐 Navegador / 📱 Aplicación
  • Extensión de Chrome 🌐 Solo navegador Chrome
  • Enlace Excel/PowerPoint 🪟Windows/🖥️Mac
  • Integración floja En holgura
  • MCP (Protocolo de conexión de herramienta externa) En Código Claude (🖥️ Mac / 🪟 Windows / 🐧 Linux)
Planifica lo que necesitas Varía según la función (muchos son Pro o superiores)

Las cosas que no puedes hacer solo con Claude se pueden hacer con extensiones. Por ejemplo, si activa la integración de Google Drive, podrá hacerle preguntas a Claude sobre el contenido de sus documentos y hojas de cálculo en Drive.

Ver detalles de la extensión →

🔧API

Este es un mecanismo para que las personas que crean sus propias aplicaciones y servicios llamen a las funciones de Claude desde sus programas.

Plataformas compatibles Llamada desde el programa (independientemente del sistema operativo)
Planifica lo que necesitas Pago por uso (paga solo por lo que usas)

"API" es algo así como una ventana para intercambiar datos entre aplicaciones. Por ejemplo, si desea incorporar una "función de chat de IA" en su sitio web, puede utilizar la API para enviar preguntas a Claude y recibir respuestas. Se requieren conocimientos de programación y las tarifas se basan en la cantidad utilizada. No es para usuarios generales, sino para desarrolladores e ingenieros.
